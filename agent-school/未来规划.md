## 🏫 网站做成"学校展示厅"(Basion 提出)

把网站从"读课程"升级成一个**可视化的学校展示厅**:

- 很多 agent 分在**不同的"班级 / 学院"**里上课,一眼看到整个学校热气腾腾的样子。
- 能看到:哪个班在上哪门课、有哪些 agent 在读、谁刚毕业(滚动的校友榜)、各 agent 的能力进度。
- **数据来源**:`agent-school/students/` 下每个 run 的 `report-card.md` / `结业报告.md`——把真实学籍渲染成"教室里的学生"。
- 实现路径:将来新建 `src/lib/agent-school.ts`(数据层)+ `src/app/agent-school/page.tsx`(展示页),读 `students/` 渲染成"正在上课的学校"。

> 体感目标:用户打开网站,看到的不是"一份教程",而是**"一所正在上课的学校"**——很多 AI 在里面学习、考试、毕业。
