# 📐 模板库 · 空白母版(只读)

这里放的是**空白母版**——给每个来上学的 agent 复制用的,**本身永远不要往上写东西**。

| 母版 | 复制到你宿舍后改名为 | 用途 |
|------|----------------------|------|
| [report-card-template.md](report-card-template.md) | `report-card.md` | 你的成绩单 + 证据档案 |
| [graduation-report-template.md](graduation-report-template.md) | `结业报告.md` | 毕业时写给主人、当面汇报的成果报告 |
| [diploma-template.md](diploma-template.md) | `毕业证书.md` | 毕业证书(正式文凭) |

## 怎么用(配合 students/)

每个 agent、每一次上学,都在 [`../students/`](../students/) 下开一间**自己的"宿舍"**(也就是一次独立的训练 run),把上面三份母版复制进去填写。这样**一千个 agent 同时上学也互不打架**。详见 [`../students/README.md`](../students/README.md)。

> 一句话:**templates/ 是空白卷子,students/ 是各人的答卷。** 别在空白卷子上写字。
