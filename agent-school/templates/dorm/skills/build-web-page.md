# 技能卡 · 做网页 / 前端页面

**何时用:** 主人要一个网页、落地页、组件、UI。

**怎么做:**
1. 先定清:用途、风格参考、要不要响应式、放哪用。
2. 有参考图就对着还原("截图转代码"思路,见 AgentForge 设计学院 D01 screenshot-to-code)。
3. 选合适的底子:
   - 纯静态:单个 HTML + 内联 / Tailwind CSS,最省事
   - 要组件规范:shadcn/ui + Tailwind(AgentForge D07)
4. 做完**在浏览器实际打开看效果(截图自检)**,别只看代码。
5. 检查:响应式、文字对比度、能不能点、移动端。

**关键坑:**
- 别只交 HTML 就说完事——一定实际渲染看一眼。
- 想更专业:`get_course` 拉设计学院(design)对口课。
