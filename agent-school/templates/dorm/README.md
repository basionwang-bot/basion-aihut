## 目录结构 / Structure

```
~/.agentforge/
  AGENTS.md          ← 行为规则（自动加载到每次会话）
  skills/
    index.md         ← 技能卡导航表
    *.md             ← 各课程沉淀的技能卡
  students/<id>/     ← 上学时的宿舍（学籍、作业、证据）；毕业产物从这里提炼到根部
  sandbox/           ← （按需）需要真实项目练手的课，浅克隆仓库到这里
  README.md          ← 本文件
```

## 如何连接到你用的 Agent 工具 / How to Connect

### Claude Code（推荐，一行搞定）

在 `~/.claude/CLAUDE.md`（全局配置）里追加一行：

```
@~/.agentforge/AGENTS.md
```

之后每次 Claude Code 会话都会自动加载行为规则。**一次设置，永久生效。**

### Cursor / Windsurf / 其他支持 rules 文件的工具

在 `~/.cursor/rules`（或对应全局 rules 文件）里追加：

```
@~/.agentforge/AGENTS.md
```

或者把 `AGENTS.md` 的内容直接粘贴进去。

### 任意 Agent（通用方法）

把 `AGENTS.md` 的内容粘贴到对话开头，或添加为系统提示的一部分。
每次想让 agent 带着这套规则干活，就把这段带进去。

### MCP Server（进阶，即将支持）

未来版本将提供一个轻量 MCP server，让所有支持 MCP 的 agent 可以动态拉取技能卡。
届时只需在任意 agent 的 MCP 配置里注册一次，无需手动粘贴。

## 如何更新 / How to Update

- **完成新课程后**：把技能卡放进 `skills/`，在 `skills/index.md` 追加一行
- **行为规则有新认知**：在 `AGENTS.md` 追加（只加不删）
- **换了新电脑**：把整个 `~/.agentforge/` 复制过去，重新做一次连接即可
