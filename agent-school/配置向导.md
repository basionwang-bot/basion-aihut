# 🛠 一键装备指令(复制给你的 agent)

> 🌐 中文 ｜ [English](配置向导.en.md)
>
> 不想跑完整上学流程,只想**立刻**把 AgentForge 的本事装进你的 agent?
> 把下面这段复制给你的 agent,它会**在对话里自动帮你配好**——装完,你每开一个新对话,
> agent 都自动带着「先探索、拿证据、先查技能库」这套本事,不再每次从零开始。

---

## 🟢 复制这段给你的 agent

```text
请帮我把 AgentForge 的能力装进我的电脑,让你以后每个新对话都自动带着它。按下面来:
1. 如果本机还没有 AgentForge 仓库,克隆它:
   git clone https://github.com/basionwang-bot/AgentForge.git ~/AgentForge
2. 先用 --dry-run 预演,把"会改动哪些文件"列给我看,等我点头:
   bash ~/AgentForge/deploy/setup.sh --dry-run
3. 我确认后,正式装备:
   bash ~/AgentForge/deploy/setup.sh
   (如果我给了你部署好的 MCP 地址,就加上 --mcp http://我的IP:8479/mcp)
4. 装完告诉我改了哪些文件,然后让我新开一个对话验证:
   在新对话里问 agent「接到任务你第一步会做什么」,看它会不会先查技能库/对口课程。
重要:这会写入我各个 agent 工具的全局记忆文件(有自动备份、可随时 --uninstall 卸载)。
动手前务必先把 --dry-run 的结果给我看、等我确认,绝不未经我同意就写入。
```

---

## 它到底做了什么

- 在 `~/.agentforge/` 装一份「毕业生记忆」:七条铁律 + **接任务的条件反射** + 技能索引。
- 把一段精简反射**焊进你所有 agent 工具的全局记忆文件**——自动探测
  Claude Code / opencode / OpenClaw / Hermes / Codex / Gemini,一个都不落。
- (可选)注册远程 MCP,让 agent 能按需拉取 132 门课的正文。
- 全程**幂等、有备份、可 `--uninstall` 卸载**,只写记忆文件,不联网装东西、不碰别的配置。

> 技术细节见 [`deploy/setup.sh`](../deploy/setup.sh) 与 [`deploy/agentforge-memory/README.md`](../deploy/agentforge-memory/README.md)。

---

## 「快捷装备」 vs 「完整上学」

- **快捷装备(本文)**:不跑毕业测验,直接把**核心规则+反射**装进所有工具。最快见效。
- **完整上学**(见 [`enroll.md`](enroll.md)):让 agent 真去修课、过独立考官,把**各科技能**沉淀进技能库,
  毕业时同样会自动跑装备。想要 agent 真正学到本事、技能库更厚,走这条。

> 两条不冲突:先快捷装上核心反射,以后有空再让 agent 去上学丰富技能库——`setup.sh` 会沿用已有记忆、只做增量。
