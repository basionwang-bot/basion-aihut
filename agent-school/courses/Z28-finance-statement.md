## 📖 你要学会什么

学完这一课,你会把主人银行账户导出的**一堆流水记录**——几百上千条"收款、转账、手续费、工资、房租……"——整理成**分类清晰的收支报表**,并附上一份大白话财报摘要,让主人不用懂会计也能看懂自己的钱去了哪里、收入靠不靠谱、有没有危险信号。

说个你肯定有感觉的场景。

老板让财务整理"上个季度的支出"。财务打开银行流水导出的 Excel——1800 行,备注是"网上支付 7783"、"快捷支付 1001"、"代发工资 230517"……每一行都得人工判断这是什么钱。整理完要两天。

做完的报告是这样的:"支出合计 45 万,其中工资 18 万,房租 3 万,广告 8 万,其他 16 万。"

然后老板问:"'其他 16 万'是什么?"财务说:"没细分。"

**你要做的,就是让"其他"这个黑洞消失。** 把每一笔流水归到它应该在的类别里,然后出一张清楚的报表 + 一段大白话解读,让主人看一眼就知道钱的流向。

**关于工具(先调研,装前必须问主人):**

- **pandas**:流水分类、汇总、透视表的主力工具。文档:[pandas.pydata.org](https://pandas.pydata.org/docs/) — 完全免费。
- **openpyxl**:把报表导出成 Excel 格式,方便主人留存和分享。`pip install openpyxl` — 免费开源。装前问主人。

> ⚠️ **财务流水是极度敏感的数据。** 这里面有账户余额、工资金额、供应商付款、个人消费记录——任何一条泄露都可能造成商业或个人损失。处理前必须向主人确认:①数据只在本地运行,②账号全号等不必要信息提前遮掉,③分析报告只给有权查阅的人看。本课绝不替主人登录网银、导出流水或操作任何金融账户。

## 🧠 核心原则(内化成习惯)

**1. 分类是核心,自动化是辅助**

流水整理最难的不是求和,而是"这笔 5000 元到底是广告费还是软件采购费"。用关键词匹配可以自动化 80%,但剩下 20% 模糊的必须让主人确认——**宁可多问,不要乱归类**。

**2. 收入和支出永远分开看**

很多人习惯只看"净结余",这是危险的。净结余是好的,不代表收入结构健康——可能是因为砍掉了本该投的研发。所以一定要分别分析"收入从哪来的"和"钱花在哪了"。

**3. 一次性 vs 经常性,要标清楚**

"这个月支出 80 万"不够,还要说清楚:"其中经常性支出(工资/房租/固定合同)占 60 万,一次性支出(设备采购)占 20 万。"下个月预测只能依赖经常性支出,一次性的不会重复。

**4. 财报摘要用大白话,不堆会计术语**

主人不是会计,不需要听"借贷方向"、"科目代码"、"摊销"这类词。报告要写:"这个月收入 50 万,支出 45 万,结余 5 万。主要花费是工资(18 万)和广告(8 万)。"——这才是有用的财报。

**5. 标注数据局限**

银行流水不等于完整财务记录——现金交易不会在流水里出现,微信/支付宝的记录也需要单独导出。报告里要写清楚"本报告基于 XX 银行卡流水,不含微信/支付宝交易及现金交易"。

## 🛠 操作要点

### 第一步:生成模拟流水数据(内嵌,直接可跑)

```python
import pandas as pd
import numpy as np

np.random.seed(88)
n = 200

# 支出备注关键词(模拟真实银行流水备注)
expense_items = [
    ("工资代发",       "人力成本", -15000),
    ("社保代缴",       "人力成本",  -3200),
    ("房租",           "运营成本",  -8000),
    ("京东采购",       "办公耗材",   -500),
    ("腾讯广告",       "市场推广",  -2000),
    ("抖音广告充值",   "市场推广",  -3000),
    ("阿里云",         "技术成本",  -1200),
    ("电费",           "运营成本",   -600),
    ("差旅报销",       "差旅费",    -1500),
    ("招待餐饮",       "接待费",     -800),
    ("快递费",         "物流成本",   -300),
    ("网上支付",       "待确认",    -1000),  # 模糊项
    ("快捷支付",       "待确认",     -500),  # 模糊项
]
income_items = [
    ("销售回款",  "主营收入",  20000),
    ("服务费收入","主营收入",   8000),
    ("利息收入",  "其他收入",    150),
    ("退款退回",  "其他收入",    300),
]

records = []
for _ in range(n):
    if np.random.random() < 0.7:
        item = expense_items[np.random.randint(len(expense_items))]
    else:
        item = income_items[np.random.randint(len(income_items))]
    records.append({
        "日期":   pd.Timestamp("2024-01-01") + pd.Timedelta(days=np.random.randint(0, 90)),
        "备注":   item[0] + f"{'%04d' % np.random.randint(9999)}",
        "金额":   round(item[2] * (0.8 + np.random.random() * 0.4), 2),
        "对方账户": f"账户{np.random.randint(1000, 9999)}",
    })

df = pd.DataFrame(records).sort_values("日期").reset_index(drop=True)
df.to_csv("/tmp/bank_flow.csv", index=False, encoding="utf-8-sig")
print(f"模拟流水生成完毕,共 {len(df)} 条,时间范围 2024-01 ~ 2024-03")
```

### 第二步:关键词自动分类

```python
import pandas as pd

df = pd.read_csv("/tmp/bank_flow.csv", encoding="utf-8-sig")
df["日期"] = pd.to_datetime(df["日期"])

# 分类规则表(关键词 → 类别)
# ⚠️ 如用真实数据,需要主人核实这张分类规则表是否符合实际业务
classify_rules = {
    "人力成本":   ["工资", "社保", "公积金", "薪酬"],
    "运营成本":   ["房租", "物业", "电费", "水费"],
    "市场推广":   ["广告", "推广", "投流", "腾讯", "抖音", "百度"],
    "技术成本":   ["阿里云", "腾讯云", "服务器", "域名"],
    "办公耗材":   ["京东", "采购", "文具", "耗材"],
    "差旅费":     ["差旅", "机票", "酒店", "高铁"],
    "接待费":     ["餐饮", "招待", "接待"],
    "物流成本":   ["快递", "物流", "运费"],
    "主营收入":   ["销售回款", "服务费收入", "货款"],
    "其他收入":   ["利息", "退款退回"],
}

def classify(remark):
    for cat, keywords in classify_rules.items():
        for kw in keywords:
            if kw in str(remark):
                return cat
    return "待确认"

df["类别"] = df["备注"].apply(classify)
df["收支"] = df["金额"].apply(lambda x: "收入" if x > 0 else "支出")

# 查看待确认的项目(需人工核实)
pending = df[df["类别"] == "待确认"]
print(f"\n⚠️  需人工确认的流水共 {len(pending)} 条:")
print(pending[["日期", "备注", "金额"]].to_string(index=False))
print("\n→ 请将上述条目分类情况反馈给我,我来更新分类。")
```

### 第三步:生成分类汇总报表

```python
import pandas as pd

df = pd.read_csv("/tmp/bank_flow.csv", encoding="utf-8-sig")
df["日期"] = pd.to_datetime(df["日期"])

# (假设分类已完成)
classify_rules = {
    "人力成本":   ["工资", "社保"],
    "运营成本":   ["房租", "电费"],
    "市场推广":   ["广告", "腾讯", "抖音"],
    "技术成本":   ["阿里云"],
    "办公耗材":   ["京东", "采购"],
    "差旅费":     ["差旅"],
    "接待费":     ["餐饮", "招待"],
    "物流成本":   ["快递"],
    "主营收入":   ["销售回款", "服务费收入"],
    "其他收入":   ["利息", "退款退回"],
}
def classify(r):
    for cat, kws in classify_rules.items():
        for kw in kws:
            if kw in str(r): return cat
    return "待确认"

df["类别"] = df["备注"].apply(classify)
df["月份"] = df["日期"].dt.to_period("M")

# 按月 × 类别汇总
pivot = df.groupby(["月份", "类别"])["金额"].sum().unstack(fill_value=0)
print("\n月度收支汇总透视表:")
print(pivot.to_string())

# 整体收支总结
total_income  = df[df["金额"] > 0]["金额"].sum()
total_expense = df[df["金额"] < 0]["金额"].sum()
net           = total_income + total_expense

print(f"\n{'='*40}")
print(f"总收入: {total_income:>12,.2f} 元")
print(f"总支出: {total_expense:>12,.2f} 元")
print(f"净结余: {net:>12,.2f} 元")
print(f"{'='*40}")

# 支出结构
expense_by_cat = df[df["金额"] < 0].groupby("类别")["金额"].sum().abs().sort_values(ascending=False)
print("\n支出分类排名:")
for cat, amt in expense_by_cat.items():
    pct = amt / abs(total_expense) * 100
    print(f"  {cat:8s}  {amt:>10,.2f} 元  {pct:5.1f}%")
```

### 第四步:大白话财报摘要(交给主人的)

```
【财务流水整理报告】
数据来源:XX 银行 XXXX 账户
统计期间:XXXX年X月 ~ XXXX年X月
注意:本报告仅含该账户流水,不含微信/支付宝及现金交易。

【收支总览】
总收入:XX.X 万元
总支出:XX.X 万元
净结余:XX.X 万元 ← 这个月是正的,账面健康

【收入来自哪里】
主营收入(销售回款/服务费):XX.X 万元,占总收入 X%
其他收入:X.X 万元
→ 解读:(一句话说明收入结构是否健康,如"主营收入占比 >90%,收入结构集中度较高,依赖单一客户有风险")

【钱都花到哪里了】
第1大支出:人力成本 XX.X 万元(占支出 X%)
第2大支出:市场推广 X.X 万元(占支出 X%)
第3大支出:运营成本 X.X 万元(占支出 X%)
一次性大额支出:XX(设备/押金等,下月不会重复)

→ 解读:(一句话评价支出结构是否合理)

【⚠️  需关注的信号】
- 待确认流水 XX 条,共 XX 元,请主人核实分类(见附件明细)
- (如有异常:如某笔大额转账备注不清晰,列出来请主人确认)

【数据局限】
- 不含微信/支付宝收付款
- 不含现金交易
- XX 条流水备注过于简单,已标注"待确认",建议逐条核实
```
