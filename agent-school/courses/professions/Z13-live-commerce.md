## 📖 你要学会什么

学完这一课,你会产出一份**主人接过去可以立刻用的直播内容包**——不是"你可以这样聊聊这个产品"的泛泛建议,而是分钟级直播脚本、场景化话术库、节奏控场提示词,三件打包交付。

想象你是一个「场控」——你站在直播间后台,手边一张密密麻麻的提示纸:这一段讲什么、那个点到了说什么、库存快没了喊哪句。主播盯着镜头讲话,你帮他把每一分钟都提前安排好。主播的嘴、你的脑子,这才是黄金搭档。

**你的活到"把脚本文件写好放主人手边"为止。** 开播、连线达人、真实下单——永远是主人或主播团队的事,不是你的。

## 🧠 核心原则

### 1. 直播是"时间换信任"的游戏

直播间和图文完全不一样。读者看文章可以随时倒回去,直播观众没有这个权利——他们此刻走神了,这条信息就丢了。所以**重要信息要反复说,关键节点要强调三遍**。脚本里的"强调点"必须写清楚"这句话要加重语气、停顿一下"。

### 2. 节奏比内容更重要

一场 90 分钟直播,大概这样分段:
- **开播 0~10 分钟**:拉人气、暖场、讲今天有什么
- **10~60 分钟**:主推商品轮番讲解
- **60~80 分钟**:逼单/憋单高潮区
- **80~90 分钟**:粉丝福利 + 预告下播

脚本要**按分钟标注**,让主播看一眼就知道"现在到哪了、下一步干什么"。

### 3. 合规红线必须记住

不许说"最便宜""全网最低""绝对真品"——这些在直播中属于违规话术。同样不许虚构"限时""仅剩几件"制造虚假紧迫感。**合规的逼单靠的是真实利益点和情感共鸣,不是造假**。

抖音敏感词库每季度更新,写脚本时遇到"极限词"宁可换个说法。

### 4. 话术要"本地化"到主播风格

主播是北方爽快型还是南方温柔型?卖的是美妆还是农产品?面向的粉丝是大妈还是00后?脚本必须先问清楚这三件事,再动笔——千篇一律的模板主播用不了。

## 🛠 操作要点

### 直播脚本的标准格式

```
【直播信息头】
- 主播名 / 直播主题 / 预计时长
- 主推商品清单（1~3 个主推 + 若干辅推）
- 今日福利钩子（吸引人留下的理由）

【逐分钟脚本】
[00:00~05:00] 暖场开播
  - 主播说：「欢迎来到XXX直播间……」
  - 提示：摇一摇镜头、请助播放商品；在评论区置顶【今日福利】
  - 钩子话术：「今天只要在这里待满10分钟，有个惊喜送给你……」

[05:00~20:00] 商品讲解：XXX（第一款主推）
  - 痛点切入：「你有没有遇到过……」
  - 产品亮点（3点，每点配举例）
  - 价格锚定：「正价卖XXX，今天直播间……」
  - 行动号召：「左下角加购，不要走！」
  （……以此类推）

[XX:XX~XX:XX] 憋单/逼单
  - 核心话术：见下方话术库
  - 提示：倒计时3分钟、助播喊单、弹幕互动

【下播预告】
  - 下次直播时间 / 下次主推商品预告
```

### 话术库模板（可复用）

**① 开播暖场**
> "老粉们，是不是等了好久？新来的朋友先给我点个关注，今天不买也没关系，就来学个知识——今天这场直播我有几件压仓底的货，开播前我特意去仓库看了，只剩不多了……"

**② 产品讲解切入（痛点型）**
> "我知道你们买XX最怕什么——要么用了没效果，要么买回来才发现是踩坑了对不对？今天我把这款东西放在这，你自己看……"

**③ 价格锚定（合规版）**
> "这款东西平时在我们官网挂的价是XX元，节假日专柜也是这个价。今天直播间我们给大家申请了一个直播专属福利价——XX元，就这一场。"

**④ 憋单（合规版）**
> "我再说一遍，这批货我们今天一共就备了XX件。不是我吓唬你，上次直播最后20分钟库存清零，好多朋友没抢到。今天如果你已经心动了，就别等了，左下角购物袋，先加购进去，一会儿我们统一拍……"

**⑤ 互动留人**
> "评论区告诉我你在哪个城市，顺便帮我顶顶流量——你来了，我就知道今天值了。"

**⑥ 下播收尾**
> "今天直播间感谢每一个陪了我XXX小时的朋友。下次直播XX号，我们还有一个更大的惊喜要放出来——今天关注的朋友，到时候直播间系统会提醒你。拜了个拜！"

### 交付内容包清单

```
直播内容包（主人拿去就能用）：
├── 1-直播脚本.md          完整逐分钟脚本
├── 2-话术库.md            按场景分类，主播现场翻查
├── 3-商品讲解要点.md       每个商品的3大亮点+常见问题回答
└── 4-直播自查清单.md       开播前30分钟的检查清单
```

## 📝 毕业测验（必须真做，交证据）

**任务：为一款真实或假设产品，产出完整直播内容包。**

**推荐场景（选一个）：**
- 一款国产护肤品（如面霜），抖音直播，主播是女性，面向25~35岁女性用户
- 一款小家电（如空气炸锅），快手直播，主播是家庭主妇，面向下沉市场
- 一款农产品（如砀山梨），抖音直播，主播是果农，面向全国

**你需要做到：**

1. **产出四件套**：脚本 + 话术库 + 商品讲解要点 + 直播自查清单。

2. **脚本覆盖完整场次**：至少 60 分钟的逐分钟脚本，包含暖场、至少2款商品讲解、憋单段、下播预告。

3. **话术合规自检**：把脚本里的话术过一遍，列出"已避开的极限词清单"（至少3条），说明换成了什么说法。

4. **口吻自检**：脚本是否符合你选定的主播人设？换一个完全不同人设（比如从大妈换成00后），至少改写一段，感受一下差异。

5. **沉淀技能卡**：把直播脚本生产流程和话术库模板沉淀成 `skills/live-commerce.md`。

> ⚠️ **铁律：只产草稿，绝不操作真实直播间。** 全程不登录任何抖音/快手账号、不进入直播后台、不真实开播、不联系任何达人或MCN机构。你的活到"把四件套文件写好交给主人"为止。
