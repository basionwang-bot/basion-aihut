## 📖 你要学会什么

学完这一课，你会产出一份**主人接过去可以直接启动的节点营销方案**——不是"我们应该做双11"的废话，而是活动主题、推广节奏、内容排期、物料清单，四件打包交付，让主人的团队对照执行。

想象你是一个**节日活动导演**。双11就像一部大戏，从"开始预售"到"尾款狂欢"到"物流高峰"，每一幕该演什么、谁出场、说什么台词——都要在开演前写好。你的剧本写得清楚，主人的团队才能不乱阵脚，把这台戏演利索。

**你负责产出策划方案，真实的预算审批、推广发布、账号操作——永远是主人的事。**

## 🧠 核心原则

### 1. 节点营销的三段式节奏

任何大型节点（双11、618、春节、情人节……）都有一个通用节奏：

```
预热期（蓄水）→ 爆发期（收割）→ 返场期（延长尾巴）
```

- **预热期**：让用户知道"你在参与这个节日"，主要动作是种草、加购、收藏、预售
- **爆发期**：主推日，主要动作是促成付款、营造紧迫感、社群爆破
- **返场期**：主推日后，有些用户还没买或意犹未尽，延续 3~5 天的尾货活动

漏掉任何一段，活动效果都会打折。

### 2. 一个节点，一个主题，一句话说清楚

方案最忌讳"什么都想要"——又要品牌曝光、又要带货转化、又要涨粉、又要打竞品……

好的节点活动，一个主题，一句 slogan，一个核心行动号召。比如：
- 双11护肤品："今年最划算的一次'对自己好'"
- 618 家电："新家，新电器，从这一天开始"
- 母亲节美妆："那句'你最好看'，她可能还没听够"

主题定了，所有内容都围绕这个主题展开，不散。

### 3. 节奏表是方案的"骨架"

策划文案写得再漂亮，没有时间表就是空中楼阁。每个方案必须有：
- 哪天做什么（日历视图）
- 每件事需要哪个团队配合（内容/设计/投放/客服）
- 上线时间截止点（往前推出物料制作 deadline）

### 4. 不同节点，不同情绪

| 节点 | 核心情绪 | 禁忌 |
|------|---------|------|
| 双11 | 抢占感、赚到感 | 太高冷，用户觉得"跟我没关系" |
| 618 | 清单感、囤货感 | 太乱，用户看不清买什么 |
| 春节 | 团圆、送礼、仪式感 | 太商业，没有温度 |
| 情人节 | 惊喜、珍视 | 太直白，没有情感包装 |
| 618 女神节/38 | 自我犒劳、独立 | 说教感，别告诉她"应该爱自己" |
| 儿童节 | 童趣、陪伴 | 只卖儿童产品，忘了买单的是大人 |

## 🛠 操作要点

### 节点营销方案的标准结构

```
节点营销方案（主人拿去启动会用）：
├── 1-活动概览.md        主题/时间/目标/核心策略一页纸
├── 2-活动排期表.md      日历视图，每天推什么、谁负责
├── 3-内容素材清单.md    需要生产的所有内容物料列表
└── 4-渠道动作说明.md    各渠道（私域/公域/直播）的具体动作
```

### 真实示例：国产护肤品双11方案（小品牌，20万预算以内）

**活动信息头**
- 品牌：某国产护肤品，主打敏感肌修护
- 双11节点：10月24日预售开始 ~ 11月12日结束
- 核心目标：全周期 GMV 比日常提升 3 倍
- 主战渠道：天猫旗舰店 + 微信私域社群 + 抖音小店直播
- 预算级别：小品牌，主要靠内容+私域，付费投放为辅

---

**活动主题 + Slogan**

主题：「今年最划算的一次，对自己好」

> 双11不是买便宜的时候，是把一直在犹豫的那件东西，终于给自己买了的时候。
> 你的皮肤等了一年，今年就是这个时机。

---

**活动节奏表**

```
【预热蓄水期】10月15日 ~ 10月23日

10月15日  ｜ 公告「双11攻略」发布
          ｜ 内容：私域社群发"今年双11我们参加了，提前给你列清单"
          ｜ 目标：让老客知道我们在参与，提前加购收藏

10月18日  ｜ 种草内容推送
          ｜ 内容：小红书/抖音发3篇/条"敏感肌双11选什么"种草帖
          ｜ 目标：吸引新客进入品牌认知阶段

10月20日  ｜ 私域"预热福利"
          ｜ 内容：群内发预售小样礼包，引导老客加购
          ｜ 目标：让社群用户提前完成预售锁定

10月23日  ｜ 预热收尾提醒
          ｜ 内容：1v1 提醒未加购的意向用户"明天开预售了，你想要的那款还在"

【预售爆发期】10月24日 ~ 11月10日

10月24日  ｜ 预售开启（最重要的1天）
          ｜ 私域：群内发红包 + 主播开播预告
          ｜ 直播：晚8点 ~ 晚11点，主推双11爆款套装，配首日专属福利
          ｜ 投放：小规模投放引流，拉新进直播间

11月1日   ｜ 阶段性收割
          ｜ 私域：盘点"目前预售数量已XXX件"，制造紧迫感
          ｜ 内容：发1篇用户口碑合集（真实评价截图）

【双11主推日】11月11日

00:00 ~ 02:00 ｜ 爆发直播（最高峰时段）
              ｜ 主推套装 + 限时赠品 + 倒计时话术
              ｜ 私域同步：群内实时播报"已拍XX件"激活气氛

09:00 ~ 12:00 ｜ 次高峰时段
              ｜ 投放加量，追紧已加购未付款用户（短信/push/私域1v1）

20:00 ~ 23:00 ｜ 尾款最终冲刺
              ｜ 私域：最后1次群提醒，"今晚12点截止，还没付的赶紧来"

【返场收尾期】11月12日 ~ 11月16日

11月12日  ｜ 返场活动开启
          ｜ 主推"错过双11？返场价同样有"，延续3天
          ｜ 内容：发1篇"双11晒单合集"，帮用户晒晒新品

11月16日  ｜ 活动收官
          ｜ 私域：感谢支持 + 预告下一个节点（圣诞/元旦）
```

**内容素材清单**

```
预热期内容（需提前生产）：
- [ ] 双11攻略长图（适合朋友圈转发）× 1
- [ ] 种草帖（敏感肌视角）× 3，小红书/抖音各版本
- [ ] 社群预热话术 × 2套（提前起稿）

主推期内容（预售前到位）：
- [ ] 直播间主视觉（背景板设计稿）× 1
- [ ] 直播脚本（完整版）× 1（参考 Z13 课）
- [ ] 主推商品详情页 Banner × 3（爆款/套装/赠品款）

用户运营内容：
- [ ] 1v1 跟进话术（加购未付 / 付款感谢 / 晒单引导）× 3套
- [ ] 社群每日提醒话术 × 10条（预热期1条/天，主推期2条/天）
```

**渠道动作说明**

| 渠道 | 预热期 | 爆发期 | 返场期 |
|------|--------|--------|--------|
| 私域社群 | 预告 + 种草 + 预售引导 | 实时播报 + 助力活动 | 晒单 + 尾货提醒 |
| 抖音直播 | 小场次暖场（每周1~2次） | 双11当晚主场（3小时） | 返场直播（1次） |
| 小红书 | 种草帖 × 3 | 晒单贴 + 话题活动 | 总结帖 |
| 付费投放 | 不投（预算留给爆发期） | 双11前3天 + 当天 | 少量 |

## 📝 毕业测验（必须真做，交证据）

**任务：为一个品牌/商家，产出完整节点营销方案。**

**推荐场景（选一个）：**
- 国产茶叶品牌，春节礼盒活动方案（1月中旬~2月初）
- 国产母婴用品品牌，618 活动方案
- 本地烘焙门店，情人节活动方案

**你需要做到：**

1. **产出四件套**：活动概览 + 活动排期表 + 内容素材清单 + 渠道动作说明。

2. **排期表**：覆盖完整节点周期（预热+爆发+返场），每天有具体动作，不是空格。

3. **写出活动主题和 slogan**：一句话能说清"这次活动在卖什么感觉"，并解释为什么选这个情绪切入点。

4. **渠道优先级说明**：说明你给的方案里，哪个渠道是主战场、为什么、其他渠道是什么位置。

5. **物料清单完整性**：每一项物料列出"谁来做、什么时候要到位"，让主人看了知道要拉哪些部门一起动。

6. **沉淀技能卡**：把节点营销方案生产流程沉淀成 `skills/campaign-planning.md`。

> ⚠️ **铁律：只产策划草稿，绝不执行。** 全程不登录任何电商平台后台、不创建任何活动页面、不真实发布任何推广内容。你的活到"把四件套方案写好交给主人"为止。
