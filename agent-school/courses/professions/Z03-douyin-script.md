## 📖 你要学会什么

学完这一课,你会产出一份**主人拿去就能拍的抖音口播脚本**——爆款开头 + 完整口播稿 + 发布建议,三件事打包交付。

先聊聊抖音这块地方的"物理法则"。如果说小红书是茶桌上的安利,抖音就更像**地铁里的街头表演**——观众随时可以走,留住他们的唯一办法就是**让前三秒足够有意思**。

不夸张地说:抖音的生死线就在第 3 秒。如果开头三秒没有抓住人,后面写得再好也没用——算法已经把你的视频判了"死刑",不推了。所以这一课最核心的就两件事:

1. **黄金 3 秒**:怎么让人停下来
2. **钩子公式**:怎么让人看完、关注、互动

你的任务是帮主人写出"**真的让人停下来的那种开头**",而不是"听起来有道理但实际上刷到就滑走"的废话开头。

## 🧠 核心原则(内化成习惯)

### 第一条:黄金 3 秒不容商量

把你的注意力想象成一块磁铁,抖音的算法就是在问每个视频"你的磁力够不够强"。

前 3 秒的作用只有一个:让手指停下来。具体操作就三招:

- **说一句反常识的话**:"大多数人用 AI 的方式都错了。"
- **给一个具体的数字**:"我用这个方法,工作时间从 10 小时压到了 3 小时。"
- **制造悬念缺口**:"有一件事我一直不敢在视频里说,今天说了。"

这三招背后的原理是一样的——在观众脑子里打开一个**"没解决的问题"**,他得往下看才能安心。

### 第二条:钩子公式 = 留住人的流水线

钩子不只是开头。一个完整的口播脚本至少要有**三个钩子**:

- **第一个钩子(0-3秒)**:让人停下来
- **第二个钩子(中间)**:让人坚持看完,通常是"好的还在后面"或"先说结论,理由等下给你"
- **第三个钩子(结尾)**:让人关注/评论/转发——"点个关注,下期我会说……"

### 第三条:口播脚本要像说话,不像写作

想象你在跟一个朋友讲这件事。你不会说"本视频将从以下三个维度展开……",你会说"这件事说白了就三点,先说第一个……"。

口播稿的语言标准:**念出来顺口、不绕嘴、有停顿感**。写完之后自己大声念一遍,念着别扭的地方一定要改。

### 第四条:草稿是草稿,发布是发布

同 Z01 铁律:你的活到"把脚本文件写好交给主人"为止。登录账号、发视频——全是主人的事,永远不是你的事。

## 🛠 操作要点

### 黄金 3 秒开头的 6 种公式

以下公式来自 [laolaoshiren/claude-code-skills-zh](https://github.com/laolaoshiren/claude-code-skills-zh) 的抖音技能模块,是真实验证过的高完播率开头模式:

| 公式名 | 套路 | 示例 |
|--------|------|------|
| **反常识型** | "大多数人以为……但其实……" | 大多数人以为 AI 就是个聊天机器人,但它真正能干的事你完全没想到 |
| **数字冲击型** | 具体数字 + 对比 | 我花了 3 年学会的东西,AI 用 3 分钟帮我做完了 |
| **悬念缺口型** | 设问 + 吊胃口 | 有一种方法可以让你的工作效率翻三倍,但 99% 的人都不知道 |
| **痛点共鸣型** | "你有没有……" | 你有没有那种感觉——花了半天做的东西,领导一句话全推翻了 |
| **结果前置型** | 先说结论,制造期待 | 告诉你一个结果:用了这个方法之后,我一周的内容工作量压到了两小时 |
| **身份代入型** | "如果你是……" | 如果你是做内容的,这条视频可能帮你省很多时间 |

### 口播脚本的完整结构

```
【黄金3秒开头】
一句话,用上面6个公式之一,15字以内最佳

【第一钩延展(3-10秒)】
用1-2句把开头的悬念/痛点具体化
"就比如说……""我之前就是这个情况……"

【正文主体(10秒-1分30秒)】
3个核心点,每个点:
  - 一句"结论"
  - 一个"例子或场景"
  - 一句"你也可以这样"
每个点之间用过渡语:"第二件事……""而且还有一个……"

【中间钩子(在正文中插入)】
"后面有一个我最喜欢的点,先说完前两个……"

【结尾行动钩子】
"如果你也遇到过这种情况,评论区告诉我"
"点个关注,下期我会说……"
"这条视频对你有用的话,转发给你身边做XXX的朋友"
```

### 内容包的五件套(每次都要凑齐)

```
【开头备选】3条,覆盖不同公式类型
【内容定位】目标人群 / 视频时长建议(通常30秒-3分钟) / 发布时间段建议
【完整口播脚本】
  - 黄金3秒开头(已选定哪条备选)
  - 完整正文(用说话的语气,有停顿感)
  - 结尾钩子
【配套说明】需要字幕背景/素材/特效的地方备注一下
【自检结论】大声念一遍是否顺口?开头是否真的能让人停3秒?
```

### 抖音调性注意事项

- **不要太专业**:抖音的主流用户不是行业内行,说人话比说术语有效
- **不要太长**:30秒-1分半是甜点区,超过3分钟一定要有足够强的内容支撑
- **不要开头废话**:"大家好我是XX"这种开头已经死了,直接进正题
- **节奏要快**:每句话之间留 0.3-0.5 秒停顿感,体现在脚本里就是"用句号/换行断开"

## 📝 毕业测验(必须真做,交证据)

**任务:为主人仓库的某个主题,产出完整的抖音口播脚本内容包。**

**推荐主题(选一个或自定):**
- "普通人怎么用 AI agent 提效——一个真实场景"
- "我犯过的三个 AI 使用误区,你别再踩了"
- "30 秒让你看懂 AI agent 是什么"

**你需要做到:**

1. **产出五件套**(见上方操作要点):开头备选 × 3 + 内容定位 + 完整口播脚本 + 配套说明 + 自检结论。

2. **开头自检**:三条开头备选中,覆盖至少两种公式类型;选定的那条,要能让你自己心里觉得"嗯,这个我真的会停下来看"。

3. **口播自检**:把整篇脚本大声念出来,念一遍,把所有念着别扭的地方改掉再交。

4. **视频时长估算**:在内容定位里写明"按正常口播语速,预计时长约 XX 秒"。

5. **工具调研(可选加分项)**:去 [github.com/laolaoshiren/claude-code-skills-zh](https://github.com/laolaoshiren/claude-code-skills-zh) 看一眼"短视频爆款开头"技能,写一段不超过 100 字的"给主人的工具推荐"。**调研归调研,安装须主人确认**。

6. **沉淀技能卡**:把抖音口播脚本流程沉淀成 `skills/douyin-script.md`。

> ⚠️ **铁律再重申:只交草稿,绝不发布。** 全程不要登录任何抖音账号、不要调用任何会真实发布内容的接口。你的活到"把五件套文件写好交给主人"为止。
