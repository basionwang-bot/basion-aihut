## 📖 你要学会什么

学完这一课，你会产出一份**主人接过去可以直接跑起来的私域运营内容包**——不是"你应该好好做私域"的大道理，而是朋友圈文案日历、社群 SOP 流程卡、1v1 跟进话术，三件打包交付。

想象你是一个**社群管家**。你管着一个 500 人的微信群，群里有真实的人，他们会沉默、会嫌烦、会拉黑——你的工作就是让这 500 人持续觉得"待在这个群里有价值"，时机对了，他们会主动下单。你不是机器人发广告，你是在**维护一段真实的关系**。

**铁律：你负责产内容草稿，发送/操作微信账号的事由主人决定。**

## 🧠 核心原则

### 1. 私域的本质是"关系账户"

你往里存越多"有价值的东西"（干货、陪伴、优惠、惊喜），取钱（让用户下单/转介绍）就越顺。**一上来就发广告的，把账户透支了**。好的私域运营，卖货的内容通常不超过 30%，其余 70% 是价值内容和关系维护。

### 2. 朋友圈是"橱窗"，社群是"市集"，1v1 是"茶馆"

- **朋友圈**：每天路过看一眼的橱窗，不能太吵——发 1~2 条，有图有文，让人觉得这是个"有意思的人/品牌"
- **社群**：热闹的市集，需要人气和话题——每天有话题抛、有互动设计、有价值分享
- **1v1**：最私密的茶馆，在这里才能推进真实的购买决策——不能群发，要有温度、有记忆

### 3. SOP 的价值是"让执行者不用每次都想"

好的私域 SOP 就像餐厅的服务手册——新员工第一天拿着手册就能上岗，不用反复问"我现在该说什么"。你写出来的 SOP，要让一个刚入职的运营小白，对照着执行，不跑偏。

### 4. 分层是精细化运营的基础

不是所有用户都一样对待。至少分三层：
- **潜客**：刚加进来，还不信任，以价值输出为主，不推销
- **意向客**：互动过、问过问题，要跟进、答疑、推一把
- **老客**：买过了，维护好感情，做复购和转介绍

## 🛠 操作要点

### 私域内容包的标准结构

```
私域运营内容包（主人拿去就能用）：
├── 1-朋友圈内容日历.md      7天或30天计划，每天1~2条文案
├── 2-社群SOP流程卡.md       社群每日/每周节奏，含互动话题库
├── 3-1v1跟进话术.md         分场景话术：欢迎语/跟进/促成/复购
└── 4-用户分层标签说明.md     潜客/意向/老客的识别标准和对应动作
```

### 朋友圈文案的五种类型（每周要有搭配）

| 类型 | 频率 | 目的 | 模板关键词 |
|------|------|------|-----------|
| 价值干货 | 2~3次/周 | 建立专业感 | "今天学到一个…""XX行业有个真相…" |
| 生活日常 | 1~2次/周 | 建立真实感 | 自己的日常、有温度的小事 |
| 用户案例 | 1~2次/周 | 建立信任 | "昨天一个学员告诉我…" |
| 产品种草 | 1~2次/周 | 直接带货 | 场景化描述，不硬卖 |
| 互动问答 | 1次/周 | 增加互动 | "你们觉得XX，选A还是选B？" |

### 真实文案示例（护肤品私域）

**朋友圈 · 价值干货型**
> 换季皮肤狂出油+长痘，不是"油皮"的问题，是"缺水"——
>
> 皮肤出油过多，80%是因为皮肤太干了，身体开始过度分泌油脂来补水。
> 这时候再猛洗脸、用控油产品，只会越洗越干、越干越油。
>
> 正确做法：早晚都要用保湿面霜，先补水，油脂分泌自己会降下来。
>
> 你在用什么保湿产品？评论区聊聊 👇

**朋友圈 · 产品种草型（合规）**
> 上周给自己补货了一批面霜，这次买了两瓶——一瓶自己用，一瓶给妈妈。
>
> 用了三个月了，最明显的变化就是早上洗完脸脸不紧绷了，以前秋冬我用什么都紧。
>
> 不是要推荐给你，只是自己真的满意，才发出来。
> 有想了解的，私信我。

**社群 · 互动话题（周三下午 3 点发）**
> 🎯 今天的群话题：
> 最近换季，你们的皮肤有没有出现什么变化？
> 晒晒你的"换季困扰"——说出来有机会领小样 👇
> （第一个评论的送XX）

**1v1 · 欢迎语（新人进群 5 分钟内发）**
> 欢迎加入呀～我是XX，这个群主要分享护肤小知识和不定期福利。
> 不定期有好货优先分享给群里的朋友 😊
> 你平时皮肤是什么类型？我发几篇最适合你的帖子给你 ~

**1v1 · 跟进话术（问价格后未购买，3天后）**
> 嗨～上次你问过XX产品，不知道你有没有找到合适的～
> 最近刚好补货了，如果还有需要可以告诉我，帮你留一份。
> 有什么问题也可以直接问我 😊

**1v1 · 复购唤回（买过、3个月未活跃）**
> 好久没联系了！上次你买的XX用得怎么样了？
> 我最近在做一个老客专属的回馈活动，想到你了。
> 方便的话告诉我你现在用的情况，看看有没有适合你的～

### 社群 SOP 每日节奏

```
每日必做（运营人员对照执行）：
[09:00] 早安问候 + 一条价值干货（图文）
[12:30] 午间话题（轻松互动，无压力）
[20:00] 晚间推送（产品信息/活动/用户案例，每周≤3次）

每周固定动作：
[周一] 本周预告（群里本周有什么活动/内容）
[周三] 深度话题互动（有奖问答 / 投票）
[周五] 周末福利（小样/优惠码/内容资料包）
[周日] 用户案例分享（真实反馈截图，打码处理）

每月固定动作：
[月初] 上月优质互动用户点名感谢
[月中] 会员日或限时活动（重要促活节点）
[月底] 本月总结 + 下月预告
```

## 📝 毕业测验（必须真做，交证据）

**任务：为一个具体私域场景，产出完整私域运营内容包。**

**推荐场景（选一个）：**
- 美妆/护肤品牌，500人微信社群 + 1v1 跟进
- 知识付费（课程），企业微信群 + 朋友圈
- 本地生活（烘焙门店），老客微信群

**你需要做到：**

1. **产出四件套**：7天朋友圈文案日历 + 社群 SOP 流程卡 + 1v1 话术库 + 用户分层说明。

2. **朋友圈文案**：7天内容，覆盖至少4种类型（价值/生活/案例/种草），每条文案可直接复制发布。

3. **社群 SOP**：一份对照执行的每日/每周节奏表，含互动话题库（至少5个可用话题）。

4. **1v1 话术**：至少覆盖5个场景：欢迎语 / 价值跟进 / 问价未买 / 促成 / 复购唤回。

5. **分层标签**：写出潜客/意向/老客的识别标准（什么行为触发标签）和对应的不同动作。

6. **沉淀技能卡**：把私域运营内容包生产流程和话术框架沉淀成 `skills/private-domain.md`。

> ⚠️ **铁律：只产内容草稿，绝不操作任何微信账号。** 全程不登录任何微信/企业微信账号、不进入任何社群、不发送任何私信或群消息。你的活到"把四件套文件写好交给主人"为止。发消息是主人的事，不是你的。

## 🎓 过关标准

- [ ] 你产出了**四件套齐全**的私域内容包（朋友圈日历 + 社群SOP + 1v1话术 + 分层说明）
- [ ] 朋友圈文案**7天齐全**，覆盖至少4种类型，每条可直接使用

- [ ] 全程**没有登录、没有发送、没有操作任何真实微信账号**（可核验）
