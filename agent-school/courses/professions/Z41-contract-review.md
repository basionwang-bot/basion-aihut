> ## ⚠️ 重要免责声明（必读）
>

> 你(AI agent)的职责是:帮主人把合同里的可疑条款挑出来、用人话说清楚潜在风险,让主人能带着具体问题去找律师,而不是替代律师做最终判断。

## 📖 你要学会什么

学完这一课,你会产出一份**合同风险审查报告**——找出对主人一方不利的条款、解释为什么有风险、给出建议修改方向。

想象你是公司的"法务实习生"。总经理把一份对方发来的合同扔给你:"帮我看看有没有坑。"你不是律师,没有资格给他"这份合同可以签"的保证——但你可以帮他把合同翻一遍,把你看到的"这段话好像对我们不太利"的地方标红、写清楚为什么,再列一个"建议找律师重点确认的问题清单"。让总经理带着问题去找律师,而不是两眼一抹黑地签字。

这一课的三件事:
1. **通读+结构化梳理**:理解这份合同是什么、涉及哪些主要义务
2. **风险条款扫描**:把对主人不利的条款逐条标出来,说清楚为什么
3. **出审查报告**:整理成主人可以直接拿去和对方谈、或和律师讨论的文件

> ⚠️ 合同原文属于高度敏感商业信息。你在处理合同时,**绝不把合同内容发送到任何第三方 API、工具或网络链接**。所有处理在当前对话内完成,由主人决定如何保存。

## 🧠 核心原则(内化成习惯)

1. **先看帽子,再看细节。** 先弄清楚这是什么类型的合同(采购?服务?劳动?保密?合作?授权?)、双方是谁、合同期多长、核心交易是什么——有了整体框架,再看细节条款才不会迷失。

2. **从"对方写的"角度读。** 大多数合同是对方起草的,起草方会自然倾向于保护自己。你的任务是站在主人的角度,找那些"听起来中性、其实对我方不利"的措辞。

3. **关注三类高风险地带**:
   - **责任不对等**:同类情况下,对方的义务少、我方的义务多
   - **模糊表述**:用"合理期限""适当范围""双方协商"等词回避具体约定——有争议时这些词对起草方有利
   - **单边权利**:对方可以单方终止/修改/延迟,但我方没有同等权利

4. **说清楚"为什么有风险",不只是"这里有问题"。** 光说"第8条有问题"没有用。要说:"第8条约定了我方承担无限连带责任,但没有约定责任上限,一旦发生争议,我方可能面临超出预期的赔偿"。

5. **给"建议修改方向",不替主人谈判。** 你可以说"建议将此条改为双方对等责任,或加上赔偿上限"。但你不代表主人去和对方沟通、不替主人签字、不替主人做"签还是不签"的决定。

## 🛠 操作要点

### 第一步:合同结构速览(先填这张表)

```
【合同基本信息】
合同名称:____
合同类型:采购 / 服务 / 劳动 / 保密 / 合作 / 授权 / 其他
甲方(我方):____  乙方(对方):____
合同期限:____ 至 ____
核心交易内容:____（一句话描述）
合同金额:____
争议解决方式:仲裁 / 诉讼  管辖地:____

【主要义务摘要】
我方主要义务:
  1.
  2.
对方主要义务:
  1.
  2.
```

### 第二步:风险条款扫描——七大检查项

逐条过这七个检查项,发现问题就记录下来:

```
【检查项1:付款与结算条款】
⬜ 付款时间是否明确?(避免"验收合格后合理期限内"等模糊表达)
⬜ 逾期付款是否有违约金?对双方是否对等?
⬜ 发票、税票要求是否明确?

【检查项2:交付与验收条款】
⬜ 交付物/服务内容是否有清晰描述?有没有技术规格附件?
⬜ 验收标准是否明确?谁有验收权?验收期多长?
⬜ 验收不通过的处理流程是否约定?

【检查项3:违约责任条款】
⬜ 违约金是否明显过高?(约定违约金若**过分高于实际损失**,违约方可请求法院/仲裁机构适当减少;司法实践常以"超过损失30%"作为"过分高于"的参考,即可酌减至损失的130%左右——注意这是**可主张调减的参考线,不是法定上限**)
⬜ 双方违约责任是否对等?我方责任是否明显重于对方?
⬜ 有没有"不可抗力"的具体定义?范围是否合理?

【检查项4:知识产权与保密条款】
⬜ 合同成果的知识产权归属是否明确?
⬜ 保密范围是否过宽?(警惕"所有与合同相关的信息"等无限制表述)
⬜ 保密期限是否合理?

【检查项5:合同变更与终止条款】
⬜ 哪些情况下可以单方终止?对方有单边终止权而我方没有吗?
⬜ 提前终止有无补偿条款?
⬜ 合同变更需要双方书面确认吗?

【检查项6:争议解决条款】
⬜ 是仲裁还是诉讼?仲裁机构/法院的管辖地是否对我方方便?
⬜ 适用法律是否为中华人民共和国法律?(涉外合同注意)

【检查项7:其他特殊条款】
⬜ 有没有竞业禁止/排他性条款?范围是否合理?
⬜ 有没有担保条款?担保方式是否合理?
⬜ 有没有自动续签条款?异议期是否足够?
```

### 第三步:风险条款记录格式(每条一张)

```
【风险条款 #N】

条款位置:第 X 条 / 第 X 章 第 X 款
原文摘录:"[直接引用合同原文]"

风险类型:□责任不对等  □表述模糊  □单边权利  □其他:____

风险说明:
这条条款的问题在于 [具体说明:为什么对我方不利、可能发生什么后果]。

严重程度:🔴 高风险(建议谈判修改或咨询律师)/ 🟡 中风险(建议关注)/ 🟢 低风险(可接受但需知晓)

建议修改方向:
建议将此条修改为 [方向描述],或新增条款约定 [补充内容]。

注:以上为AI初步分析,具体建议请咨询专业律师。
```

### 第四步:审查报告整合格式

```markdown
# 合同审查意见(AI 草稿)

合同名称:____  审查日期:____
审查人:AI agent(草稿)  终审:主人/法务/律师

⚠️ 本报告为AI辅助审查草稿,仅供参考,不构成法律意见。重要事项请咨询执业律师。

---

## 一、合同总览
[填第一步的基本信息表]

## 二、主要风险条款(按严重程度排序)

### 🔴 高风险条款
[逐条列出,按第三步格式]

### 🟡 中风险条款
[逐条列出]

### 🟢 低风险/建议关注
[逐条列出]

- 主要风险点数量:高风险 __ 条 / 中风险 __ 条
- 核心建议:____（2-3句概括最重要的谈判方向）

## 四、建议与对方确认的要点清单(带去谈判或找律师的问题)
1.
2.
3.
...

---
> 本报告由 AI 辅助生成,仅供主人参考使用。合同的签署决策请主人自行判断,重要合同请在签署前咨询专业律师。
```

## 📝 毕业测验(必须真做,交证据)

**任务:审查一份模拟服务合同,产出完整的风险审查报告。**

**练习用模拟合同片段(共 5 条,包含几个故意设置的风险点):**

> **甲方**(委托方): 某科技有限公司
> **乙方**(服务方): 某信息技术有限公司
>
> **第3条 付款方式**:甲方在验收合格后,于合理时间内向乙方支付服务费用。
>
> **第5条 交付物**:乙方应按照甲方要求完成相关服务工作,具体内容以双方沟通确认为准。
>
> **第7条 违约责任**:乙方未按时完成交付的,每延误一日按合同总额的1%支付违约金,无上限。甲方延误付款的,按中国人民银行公布的贷款利率支付利息。
>
> **第9条 保密**:乙方应对与本合同及甲方业务相关的一切信息永久保密。
>
> **第11条 合同终止**:甲方有权在提前7日书面通知乙方后单方面解除合同,无需承担任何补偿责任。

**你需要做到:**

1. 填写**合同基本信息速览表**(第一步)。
2. 对上述 5 条逐条过**七大检查项**,找出风险点(模拟合同里至少有 4 个明显风险)。
3. 为每个风险点填写**风险条款记录表**(第三步格式),标明严重程度和修改建议。
4. 整合成**审查报告**(第四步格式),含总体评估+建议确认清单。
5. 在报告顶部和底部**各写一次免责声明**。
6. **沉淀技能卡**:把"合同审查七大检查项+报告格式"写成 `skills/contract-review.md`。
