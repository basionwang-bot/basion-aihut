## 📖 你要学会什么

学完这一课,你会产出一份**主人可以直接用于视频号的内容包**——图文帖子草稿 + 短视频脚本 + 微信生态分发建议,三件事打包交付。

先说一件重要的实话。

**视频号目前没有成熟的专门 AI 技能。** 和公众号有 wewrite 和 md2wechat、小红书有归藏卡片和 xhs-studio 不同,视频号的专门工具几乎是空白地带。这是 2026 年的现实情况,不是我藏着掖着——告诉你真实状况比给你一堆没用的工具更重要。

但这不妨碍你产出好内容。视频号的核心价值不是"有多少 AI 工具帮你做",而是**微信生态的独特社交分发方式**——你的内容是通过朋友点赞、转发到朋友圈、微信群分享来流传的,这是和抖音、小红书完全不同的传播逻辑。

搞懂这个逻辑,你就知道视频号的内容怎么写了。

**关于现成工具(先调研,装前必须问主人):**

- **cn-content-matrix**:把一个主题改写成各平台原生口吻,可以用于把公众号内容迁移成视频号图文风格。链接:[github.com/fullstackcrew-alpha/skill-cn-content-matrix](https://github.com/fullstackcrew-alpha/skill-cn-content-matrix)
- **humanizer(去 AI 腔)**:视频号内容更依赖"人味",这个工具帮你把 AI 写作腔调洗掉。链接:[github.com/blader/humanizer](https://github.com/blader/humanizer)
- **Brand Voice(官方)**:Anthropic 官方出品,固化品牌调性、统一口吻——微信生态的主人形象一致性很重要。链接:[github.com/anthropics/knowledge-work-plugins](https://github.com/anthropics/knowledge-work-plugins)

> ⚠️ 这些工具调研归调研,**装之前必须先征得主人确认**。本课重点是产出内容草稿,绝不替主人登录或操作任何真实账号。

## 🧠 核心原则(内化成习惯)

**1. 视频号的"地形":社交驱动,不是算法驱动**

抖音和小红书是算法给你推内容——你的内容如果能留住陌生人,算法就把你推给更多陌生人。

视频号不一样。它嵌在微信里,更依赖**社交关系**的传播:你的朋友点了赞,你在朋友圈才会看到;你的内容被微信群转发,才会大量传播。

这意味着:视频号的内容要让**你认识的人愿意转发给他们认识的人**。比起让陌生人停下来,更重要的是让老朋友觉得"这个值得分享"。

**2. 视频号的内容调性:有温度、实用、不装**

微信是一个强关系社交网络,人们在这里分享的东西通常是"有用的"或"有感触的"或"代表我立场的"。

好的视频号内容通常有这几种感觉:
- 实用干货型:"这三个方法真的有用,存一下"
- 观点共鸣型:"说出了我一直想说的话"
- 情感触动型:"看完会想转给某人看"

**3. 图文 vs 短视频:各有用场**

视频号同时支持图文帖子和短视频。你需要根据主人的实际情况建议用哪种:
- **图文帖子**:生产成本低,适合快速发布、知识分享、日常内容;在朋友圈转发效果好
- **短视频**:传播力更强,适合有明确主题的内容;但需要主人真的去录制

**4. 草稿是草稿,发布是发布**

同 Z01 铁律:你的活到"把内容包写好交给主人"为止。登录账号、发布内容——全是主人的事,永远不是你的事。

## 🛠 操作要点

### 视频号图文帖子的结构

视频号图文帖子比公众号短很多,但比微博长一点。把它想象成**一个精炼的朋友圈长文**:

```
【开头一句话】
最重要的观点或最让人有感触的话放第一行
（这是在朋友圈展开前唯一可见的内容）

【正文主体】
3-5段,每段2-4行
有具体场景或例子
语气口语、有温度

【结尾】
一句总结 或 一个引发共鸣的问题
可以加 #话题标签（1-3个,不要太多）
```

### 视频号短视频脚本结构

视频号短视频的最佳时长是 1-3 分钟,太长了朋友圈滑过去了。

```
【开头(0-5秒)】
不要打招呼,直接说核心——"今天要给你分享一件事"
或者直接说结论——"用了这个方法之后,我发现……"

【正文(5秒-2分30秒)】
讲一件事,讲清楚
有具体数字/场景,不要空泛

【结尾(最后10-15秒)】
一句话总结
可以说"觉得有用的话分享给朋友"/不说也行
```

### 内容包的五件套(每次都要凑齐)

```
【内容形式建议】图文还是短视频?理由是什么?
【内容定位】目标人群 / 适合在哪些微信群或朋友圈触达
【图文帖子草稿】（无论选哪种形式都要写）
  - 开头一句话(最重要的观点/感触)
  - 正文
  - 结尾+话题标签
【短视频脚本草稿】（如果建议做视频的话）
【分发建议】适合通过哪些渠道进入微信生态(朋友圈/群分享/公众号联动)
【自检结论】这条内容有没有让你想转给某个人?如果没有,是哪里出了问题?
```

### 微信生态的分发逻辑

视频号内容最有效的分发路径通常是:

```
公众号文章 → 文章末尾引导到视频号
视频号短视频 → 转发到朋友圈 / 微信群
微信群里的分享 → 引发二次传播
```

如果主人已经有公众号(参考 Z01 课),可以建议公众号和视频号联动:
- 公众号写深度文章 → 视频号做 1-2 分钟精华版
- 视频号做话题视频 → 公众号做配套深度文章

### 诚实说明:视频号工具的现状

截至 2026 年,视频号的专门 AI 创作工具几乎空白:
- 没有像 wewrite 那样的全流程技能
- 没有像归藏卡片那样的专属配图工具
- 发布也没有稳定可用的 API

**能用的通用方案**:
- 内容写作:用 cn-content-matrix 把已有内容迁移成视频号风格
- 去 AI 腔:用 humanizer 润色
- 配图:借用归藏卡片([op7418/guizang-social-card-skill](https://github.com/op7418/guizang-social-card-skill))生成图文卡,尺寸选 1:1 或 3:4
- 发布:只能手动,无法 AI 自动化

这个空白反过来也是机会——等主人学会了模块六,说不定可以自己填这个坑。

## 📝 毕业测验(必须真做,交证据)

**任务:为主人仓库的某个主题,产出完整的视频号内容包。**

**推荐主题(选一个或自定):**
- "用 AI 做内容的人,和不用 AI 的人,差距正在被这件事拉开"
- "我用 AI agent 做了一周内容之后发现的三件事"
- "给身边想用 AI 但不知道怎么开始的朋友"

**你需要做到:**

1. **产出五件套**(见上方操作要点):内容形式建议 + 内容定位 + 图文帖子草稿 + 短视频脚本草稿 + 分发建议 + 自检结论。

2. **图文帖子草稿**:开头一句话必须是让人想继续读或想转发的那种——不是"今天分享一下……"。

3. **自检**:完成后问自己"如果我是主人的朋友,看到这条内容会转发给谁?"——如果回答是"不会转发给任何人",说明内容还不够好,继续改。

4. **工具现状说明**:在内容包里写一段(50-100字)"给主人的诚实说明":视频号专用工具现状 + 当前可用替代方案。

5. **沉淀技能卡**:把视频号内容包流程沉淀成 `skills/shipinhao-content.md`。

> ⚠️ **铁律再重申:只交草稿,绝不发布。** 全程不要登录任何视频号账号、不要操作任何微信相关系统。你的活到"把五件套文件写好交给主人"为止。
