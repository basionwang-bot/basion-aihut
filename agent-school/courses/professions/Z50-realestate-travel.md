## 📖 你要学会什么

学完这一课,你会产出两种"让人心动"的文案:**房源描述文案**让买家/租客看了想立刻约看房,**旅游行程文案**让读者看了就想订机票。

两种文案表面上很不同,但底层是同一件事:**用具体的细节打动人,而不是用形容词堆砌**。

先说个对比。

**房源文案的典型错误写法:**"精装修,采光好,交通便利,拎包入住"——这八个字几乎可以贴在所有房子上,等于什么都没说。**打动人的写法是:**"早上七点,阳光从东南角的落地窗漫进来,正好打在那张白色的双人床上——这套房子,清晨是有颜色的。"

**旅行文案的典型错误写法:**"Day 1:上午故宫,下午颐和园,晚上品尝北京烤鸭"——这是行程表,不是文案。**打动人的写法是:**"进故宫之前先去角楼拍一张倒影——清晨护城河水平如镜,金色琉璃瓦和它的倒影几乎分不清真假。"

这门课教你掌握这个差距。

> 铁律写在最前面:你只出草稿。绝不替主人发布到任何平台、不操作任何账号、不上传任何真实房源或订单。

## 🧠 核心原则(内化成习惯)

**1. 细节是打动人的最小单位**

"采光好"不打动人,"从窗台能看见对面小学的操场,每天早上有孩子做早操"才打动人。

"风景美"不打动人,"走到悬崖边,脚下是云海,远处的山尖刚好露出一点——你会忘记自己还没吃早饭"才打动人。

原则:**每写一个形容词,问自己"能换成一个具体画面吗?"能换就换。**

**2. 房产文案:卖的是"在这里生活的样子"**

买房或租房的本质决策是:"我能不能想象自己住在这里?"

所以最有效的房产文案,不是描述建材和设施,而是**描述生活场景**——早上起来干什么,周末去哪里,孩子回家之后……把这些场景写出来,比任何平方米数据都有说服力。

**3. 旅游文案:卖的是"去了之后你会变成什么样的人"**

旅行营销的底层是一种向往——"去了丽江,我就能拥有那种慢下来的状态"。旅行文案要触碰的,不是景点本身,而是**读者对旅行之后那个自己的期待**。

所以好的旅游行程文案里,除了"去哪里、几点",还要有**情绪时刻**:"走进小巷拐角突然看见一家只有三张桌子的小店,老板是个 70 岁的婆婆,她会给你倒一碗米酒,然后用带口音的普通话问你从哪里来。"

**4. 两种文案都需要"锚点信息"保证可信度**

纯粹的情感描写容易让人觉得"是不是在吹牛"。所以打动人的文案往往**情感描写和锚点信息交替出现**:
- 房产:一段场景描写 + 一个具体数据(地铁 3 分钟 / 楼龄 5 年 / 小区绿化率)
- 旅游:一段情绪描写 + 一个实用信息(开放时间 / 门票价格 / 推荐季节)

**5. 知道你的受众在哪个平台看**

- 贝壳/安居客:结构化信息为主,文案在"亮点"字段里放
- 小红书:竖屏短笔记,情绪浓,句子短,emoji 适量
- 微信/朋友圈:熟人关系,口语化,分享感强
- 马蜂窝/携程攻略:长篇详细,实用信息多,图片导向

## 🛠 操作要点

### 房源描述文案的四层结构

```
第一层:钩子句(1-2句)
  — 不说"精装三居室",说"站在主卧窗边往下看,楼下那排梧桐今年秋天特别黄"
  — 目的:让买家/租客产生"我想亲眼看看"的冲动

第二层:生活场景(3-5句)
  — 早上/周末/孩子放学/下班后……具体的生活画面
  — 每个场景绑定一个实际优势(朝向/周边商圈/学区/楼层)

第三层:硬信息打包(可用列表)
  — 面积/楼层/户型/装修年份/车位情况/物业费/周边配套
  — 这层要准确,主人需要核实数字

第四层:行动号召
  — 明确的看房邀约:"随时可约看房,工作日下午/周末全天均可"
  — 联系方式占位符(主人填入)
```

### 旅游行程文案的三种层次

**层次一:信息型行程单**（给已决定出行的旅伴看）
- 格式:Day X → 上午/下午/晚上 + 地点 + 实用贴士
- 风格:简洁准确,重点标注交通/费用/注意事项

**层次二:种草型行程文案**（发小红书/朋友圈,目的是让别人也想去）
- 格式:以情绪时刻为节点串联行程,穿插实用信息
- 风格:第一人称分享感,有惊喜发现,有情绪起伏

**层次三:营销型行程文案**（旅行社/民宿/景区用）
- 格式:海报标题 + 亮点三条 + 行程概览 + 行动号召
- 风格:专业感 + 情感共鸣,重点突出差异化体验

**示例 A:北京东三环某二手两居室(贝壳平台)**

```

这套房子最打动我的不是装修——是从饭厅往外看,正好能看见对面小区
的那片银杏,每到深秋整片都是金黄色,城里不多见的景。

【生活场景】

【硬信息】
· 面积:87㎡ / 两室一厅一卫
· 楼层:8/18 层,南北通透
· 装修:2019 年精装,厨房电器在
· 小区:XX 年代建成,24 小时门卫,地下停车位可租
· 物业费:约 X 元/月(主人核实)

【行动号召】
随时可预约看房。工作日 17:00 后或周末全天均可——看房不占你多少时间,
进门转一圈心里就有数了。联系方式:[主人填入]
```

**示例 B:云南大理三日游·种草型小红书文案**

```
【标题备选】
✨ 大理三天,我终于理解了为什么那么多人「住下来不走了」
🌸 大理自由行避坑指南｜这几个地方千万别错过
☁️ 去了一次大理,我把下次的机票也定了

【正文草稿】

去大理之前,我以为我只是来"凑个热闹"。

Day 1·下关飞机场 → 古城

落地已经快六点了。没着急赶古城,先在酒店放下行李,
推开窗——苍山就在那里,一点不客气地扑过来。
晚上随便走进古城一家看起来灯光温暖的小馆子,
老板推荐了乳扇烤着吃,焦香焦香的,蘸一点辣椒面——
那天晚上我吃了三串。

Day 2·双廊 → 沙溪古镇

双廊是必去的。但有一件事我没料到:
早上六点半去才对——游客还没来,洱海上有薄雾,
湖边晨跑的当地人会对你点头微笑。
那种宁静感,是中午人山人海的双廊给不了的。

下午坐包车去沙溪(大概 1.5 小时),建议住一晚。
四方街天黑之后只剩本地人出来遛弯——喝一瓶啤酒坐在台阶上,
真的有种"时间慢下来"的错觉。

Day 3·苍山 · 大理古城返程

苍山门票 72 元(2024 年价格,主人核实最新),
可以索道上去、徒步下来——膝盖没问题的话强推。
下来之后在古城的洋人街随便逛,
最后一顿饭一定要吃一碗过桥米线。

【实用贴士】
· 最佳时间:3-5月洱海花开 / 9-11月苍山红叶
· 住哪里:古城内小院民宿(300-600元/晚)/ 双廊洱海边（贵但值）
· 交通:下关机场 - 古城 打车约 40 元 / 包车大理全程推荐(灵活)
· 避坑:洱海骑行路段人特别多,如果要骑建议早上 7 点前出发

【配图建议】
1. 苍山倒映洱海的画面(早晨拍最佳)
2. 双廊早晨空镜(横图,清冷感)
3. 沙溪四方街夜景
4. 乳扇/过桥米线食物特写
5. 民宿院子内一角(有院、有花、有蓝天)

【话题标签】
#大理旅游 #云南旅行 #大理古城 #沙溪古镇 #洱海 #旅游攻略 #打卡地推荐
```

**赛道 A:房产**

- 刚需两居室(目标客户:在附近上班的年轻夫妻)
- 改善型三居室(目标客户:有孩子的中产家庭)

每份文案需包含四层结构(钩子句 + 生活场景 + 硬信息 + 行动号召),硬信息处用占位符标注"(主人核实)"。

**赛道 B:旅游**

- **信息版**(行程单口吻):Day X 结构,含时间/地点/费用/交通/注意事项

**两条赛道都做(加分项)**:分别各出一份,展示两种文案思路的切换能力。

**共同要求:**

1. 每份文案都要有**至少 2 处具体细节描写**——不是形容词,是画面。

2. **自检三问**:
