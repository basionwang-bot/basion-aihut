## 📖 你要学会什么

学完这一课，你会产出一份**品牌定位文档 + 口吻文案包**——不是"我们的品牌价值观是创新、品质、用户至上"这类虚话，而是一句话品牌定位（能刻在门头上的那种）、品牌口吻说明书、以及各渠道的实际文案示例，全部打包交付。

想象你是给一家餐厅做"形象设计"的人。这家餐厅菜做得不错，但到了美团上，名字叫"张记餐馆"，简介写着"欢迎光临，品质保证"——路过的人根本分不清它和隔壁"李记饭店"有什么不一样。你的工作就是帮他**找到一句话说清"我是谁、跟别人有什么不同"**，然后让这句话贯穿进每一个接触点的文字里。

**你负责产出品牌定位文档和文案草稿，最终决策和发布由主人来。**

## 🧠 核心原则

### 1. 定位是"在用户脑子里占一个位置"

《定位》这本书的核心观点：人的脑子里对每个品类只能记住 2~3 个品牌。可口可乐是"快乐汽水"、云南白药是"止血的"、瑞幸是"便宜的好咖啡"——这些都是在消费者脑子里占住的"格子"。

一个品牌的定位工作，就是回答：**在我这个品类里，我想在用户脑子里占哪个格子？** 这个格子现在是空的吗？如果不空，我能比现在占着的那个品牌说得更清楚吗？

### 2. "一句话定位"必须能通过三个测试

**Test 1 · 完形填空测试**
> "当用户想要____（具体需求），他们应该想到你，因为你是____（独特价值）。"

**Test 2 · 对比测试**
> "我们不像____（竞品），我们是____（差异点）。"

**Test 3 · 相信测试**
> 这句话用户信吗？有没有什么证据支撑？（如果只是自夸，用户不会信。）

三个测试都过了，这句定位才算成立。

### 3. 口吻比内容更重要——但很难量化

两个品牌说同一件事：

A品牌："本品精选优质原料，经过严格质量检测……"  
B品牌："给你的，只挑最好的。"

说的是同一个意思，但感受完全不同。A 在自说自话，B 在跟用户说话。

口吻（Brand Voice）就是"这个品牌如果是一个人，他说话是什么风格"——是专业理性的老教授，还是幽默接地气的邻居大哥，还是安静温柔的知心姐姐？口吻一旦定了，所有文案都要守住这个人设不跑偏。

### 4. 定位和口吻是"内核"，内容是"外衣"

定位和口吻可以几年不变，但内容每天都在更新。好的品牌，翻遍三年的微博/公众号，每一条都像同一个人说的——这才是真正守住了品牌口吻。

## 🛠 操作要点

### 品牌定位文档的标准结构

```
品牌定位文档（主人内部使用 + 对外输出的底稿）：
├── 1-定位一页纸.md        核心定位句 + 三个测试结果 + 支撑理由
├── 2-品牌口吻说明书.md    口吻词汇表 + 正反例子（该说什么/不该说什么）
├── 3-渠道文案示例包.md    公众号简介/小红书签名/天猫首页文案/slogan ×3
└── 4-品牌故事一段话.md    适合"关于我们"页面的品牌故事（200字以内）
```

### 定位工作流（四步走）

**第一步：摸清"我是谁"**
```
回答这五个问题（主人填，你来整理）：
① 我们卖什么？（品类，越细越好）
② 主要客户是谁？（越具体越好，"25~35岁有孩子的城市女性"比"年轻女性"好）
③ 他们在这个品类里最大的痛点/渴望是什么？
④ 市场上主要竞争对手是谁？他们在传播什么？
⑤ 我们跟他们相比，有什么是我们做得到、他们做不到（或者说不好意思说）的？
```

**第二步：提炼"一句话定位"**

用公式：`[目标人群] + [场景/需求] + [差异化价值]`

示例：
- "给敏感肌宝宝的第一瓶面霜——专研刺激源，只留最干净的成分。"
- "在外卖和自己做饭之间，还有一个选择：15分钟做出一顿真正的饭。"

**第三步：写口吻说明书**

```
品牌人设：这个品牌如果是一个人，他是谁？
（例：一个35岁的妈妈，有两个娃，曾在宝洁做过产品研发，说话理性不煽情，
 但真的关心你孩子，会把成分表给你分析到位，不会骗你买贵的没用的东西。）

口吻关键词（3~5个）：
  - 专业但不高冷
  - 真实、不夸张
  - 关心，但不啰嗦
  - 有温度，不煽情

正确示范（Do）：
  ✅ "这款面霜里没有香精——不是因为香精贵，是因为宝宝皮肤不需要。"
  ✅ "我们做过测试，结果没有你想的那么完美，但够真实。"

禁止示范（Don't）：
  ❌ "我们是最专业的妈妈品牌！"（夸大且空洞）
  ❌ "天下妈妈心连心……"（煽情，假）
  ❌ 每条文案都出现"宝宝"这个词（说话方式不像真人）
```

**第四步：落地到各渠道文案**

一套定位，翻译成各渠道的具体文字：

| 渠道 | 字数限制 | 文案任务 |
|------|---------|---------|
| 微信公众号简介 | 120字 | 让第一次看到的人明白你是做什么的 |
| 小红书账号简介 | 100字 | 让人想关注你 |
| 天猫/京东店铺首页大字 | 20字以内 | 一句话抓住访客 |
| Slogan（对外主打语） | 10字以内 | 能记住、能传播 |
| 关于我们（品牌故事） | 200字 | 让人觉得这个品牌有温度有来历 |

### 真实示例：国产儿童护肤品牌「小净」（假设品牌）

**一、定位一页纸**

目标人群：25~38岁的城市妈妈，有0~3岁宝宝，有基础成分认知（会看成分表），不轻信广告。

核心需求：给宝宝用的东西，要干净（成分少而精）、有效（护屏障）、好用（不拔干不过油）。

主要竞品传播方向：
- 日本品牌：强调温和历史、日本标准
- 国内大品牌：强调妈妈口碑、价格亲民
- 小众进口品：强调专研某肤质

我们的差异：深度成分透明（每款产品公开成分逻辑）+ 国内皮肤科合作验证

**一句话定位：**
> "给宝宝皮肤的第一瓶面霜，成分只保留必须的——我们解释每一种加进去的理由，和每一种我们不加的理由。"

完形填空：当妈妈想给宝宝选一款成分干净、能信任的面霜，她们应该想到小净，因为小净是唯一一个把"为什么加这个成分"和"为什么不加那个成分"都公开说清楚的国产婴童护肤品牌。

对比测试：我们不像其他婴童品牌只强调"温和"，我们解释"温和是怎么做到的"。

相信测试：有皮肤科医生背书 + 每款产品成分说明书可下载 → 可信。

---

**二、品牌口吻说明书**

品牌人设：一个学化工的年轻妈妈，现在自己做了儿童护肤品，对成分很懂，不追网红成分，说话直接不绕弯，关心用户但不煽情，会跟妈妈们像朋友一样聊成分，但不替妈妈做决定。

口吻关键词：**透明 / 理性 / 有温度 / 不废话**

| 口吻维度 | 正确示范 | 禁止示范 |
|---------|---------|---------|
| 说成分 | "加了燕麦β-葡聚糖，因为它能帮皮肤屏障修复——不是因为它贵，是因为它有效" | "精选天然植物精华，倍加呵护" |
| 说效果 | "测试结果显示，80%宝宝用后皮肤泛红减少——不是100%，因为宝宝的肤质不一样" | "一用见效！宝宝皮肤明显改善！" |
| 说价格 | "这款比上款贵30元，因为多了一步工艺——你决定要不要" | "超高性价比！不贵！" |
| 跟妈妈沟通 | "你可以直接问我为什么加这个，我给你讲清楚" | "妈妈们放心，我们都是为了宝宝好！" |

---

**三、渠道文案示例包**

**公众号简介（100字）**
> 儿童护肤成分研究室。我们每款产品都有一份《成分说明书》——告诉你每种成分加进去的理由，和我们没加某些成分的理由。你来看，你来决定。欢迎来这里问任何成分问题，我们都认真回答。

**小红书账号简介（80字）**
> 做儿童护肤品的，化工背景。在这里分享成分知识和我们品牌做产品的思路。不卖焦虑，只讲成分。你有任何成分问题，欢迎评论区@我。

**天猫店首页主标语（18字）**
> 宝宝皮肤用的每一种成分，我们都有理由。

**Slogan 备选 × 3（10字以内）**
> A. 少加一点，可以更好。  
> B. 每个成分，我们都能解释。  
> C. 干净，是最好的配方。

**品牌故事（180字）**
> 小净从一个问题开始：为什么婴儿护肤品的成分表越来越长？
>
> 我是一个有两个孩子的妈妈，也学过化工。当我去看市面上的婴儿护肤品，我发现：很多"温和"产品里，其实加了很多宝宝根本不需要的成分——香精、过多的防腐剂、流行成分……加进去的理由是"消费者喜欢"，不是"宝宝皮肤需要"。
>
> 于是我们从 2020 年开始做小净：每款产品只加宝宝皮肤真正需要的成分，每一种都有理由，每一种不加的也有理由。成分说明书免费下载，我们没什么好藏的。

## 📝 毕业测验（必须真做，交证据）

**任务：为一个真实或假设品牌，产出完整品牌定位文档。**

**推荐场景（选一个）：**
- 国产手工皮具小品牌，面向 30+ 有购买力的男性
- 国产家庭自制零食品牌，面向有孩子的宝妈
- 本地独立咖啡馆，面向周边写字楼白领

**你需要做到：**

1. **产出四件套**：定位一页纸 + 口吻说明书 + 渠道文案示例包（至少4个渠道）+ 品牌故事（200字）。

2. **定位一页纸**：填完完形填空 + 对比测试 + 相信测试三个，并给出通过/未通过的判断及理由。

3. **口吻说明书**：给出至少 3 组"正确示范 vs 禁止示范"对比，覆盖不同沟通场景。

4. **Slogan**：至少 3 条备选，说明每条的情感侧重点不同在哪里。

5. **差异化验证**：说明你给的这个定位，市场上现在有没有竞品在占这个位置——如果有，你怎么说得比他们更清楚？

6. **沉淀技能卡**：把品牌定位生产流程和口吻框架沉淀成 `skills/brand-positioning.md`。

> ⚠️ **铁律：只产定位草稿，绝不替品牌发声。** 全程不登录任何品牌账号、不真实发布任何品牌内容、不注册任何商标或域名。你的活到"把四件套文件写好交给主人"为止。品牌对外发声是主人的决定，不是你的。
