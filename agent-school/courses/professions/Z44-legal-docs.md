## 📖 你要学会什么

学完这一课,你会帮主人起草**常见商业法律文书的模板**——侵权告知函、版权声明、免责声明、合作备忘录这类文书——让主人手里有一份结构正确、措辞到位、可以拿给律师"微调后即用"的草稿,而不是从零开始。

**重要免责声明(必读,必须向主人转达):**

> ⚠️ **AI 产出的法律文书草稿仅供参考,不构成法律意见,不具备正式法律效力。** 法律文书的效力取决于具体事实、适用法律和司法实践,**正式使用前请务必经专业律师审核确认**。

想象你是一个熟练的文书助理。律师事务所接了一个新客户,律师大概说了个思路,你负责把空白模板填起来、把基本结构搭好——格式规范、要素齐全、语言书面——然后律师看一眼、微调几处、就能发出去。你不是律师,但你能让律师省下一半时间。这就是这一课要培养的能力。

## 🧠 核心原则(内化成习惯)

1. **法律文书有固定"骨骼",不许随意重排。** 每种文书都有约定俗成的格式——标题、发件方、收件方、正文、落款、日期。就像一张正式请假条必须写清楚"请假人/请假事由/起止日期/负责人签字"一样,法律文书的要素缺一不可。缺了,文书的效力就打折扣。

2. **事实陈述要清晰、具体,不要含糊。** "对方侵权了"不够——要写清楚"对方于何时,在哪个平台,以何种方式,侵犯了我方哪项权利(如著作权第X条)"。事实越具体,文书越有分量。

3. **诉求要明确,给对方留出合理期限。** 一封告知函如果只说"你侵权了,请你注意",对方完全可以当没看见。要写清楚"请于收到本函 X 个工作日内完成以下操作:……"——这既是礼貌,也是法律程序上的必要前置。

4. **语气书面而不要带情绪。** 法律文书不是投诉信,不要写"你们的行为极其恶劣"——写"上述行为已构成对本公司著作权的侵犯,依据《著作权法》第X条,本公司依法享有追索权"。书面、克制、有依据。

5. **每份文书都要标明"草稿"和免责声明。** 在文书页眉或页脚标注:"本文书草稿由 AI 辅助生成,正式使用前请经专业律师审核。"

## 🛠 操作要点

### 常见文书类型速查

| 文书类型 | 典型使用场景 | 核心要素 |
|----------|-------------|----------|
| 侵权告知函 | 发现对方抄袭内容/仿冒商标/盗用图片 | 侵权事实+法律依据+具体诉求+期限 |
| 版权声明 | 网站/App/公众号的版权保护说明 | 权利归属+禁止行为+授权方式+联系方式 |
| 免责声明 | 产品/服务/内容的责任边界声明 | 服务范围+不保证条款+责任限制+适用法律 |
| 合作备忘录(MOU) | 合作初期的意向确认 | 双方信息+合作范围+权利义务+保密条款+有效期 |
| 授权使用书 | 授权他人使用自己的内容/图片/代码 | 授权方/被授权方+授权内容+使用范围+期限+费用 |

### 侵权告知函 · 标准模板框架

```
[文书标题]
关于侵犯【本公司/本人】著作权的告知函

致:【被告知方名称】

一、发函方信息
发函方:【公司名称/个人姓名】
地址:【联系地址】
联系方式:【邮箱/电话】

二、侵权事实
【被告知方】于【时间】在【平台/渠道/链接】发布/使用了【描述侵权内容】。
上述内容系本方于【创作/发布时间】原创的【作品类型】(以下简称"涉案作品"),
涉案作品的著作权归本方所有,本方对此保留完整证据。

三、法律依据
依据《中华人民共和国著作权法》第【X】条,上述行为已构成对本方著作权的侵犯。

四、诉求
请贵方于收到本函【10】个工作日内:
1. 立即停止上述侵权行为;
2. 删除/下架涉案内容;
3. 就上述侵权行为向本方书面致歉;
4. 赔偿本方经济损失【人民币 X 元】(或就赔偿金额与本方协商)。

逾期未回复或未采取行动,本方将依法向有关部门投诉或提起法律诉讼,届时一切法律后果及费用由贵方承担。

五、落款
发函方:【签名/盖章】
日期:【年】年【月】月【日】日

---
⚠️ 本草稿由 AI 辅助生成,仅供参考,正式发函前请经专业律师审核。
```

### 免责声明 · 标准模板框架

```
[文书标题]
【产品/服务/网站名称】免责声明

一、信息准确性
本【产品/平台/内容】所提供的信息仅供参考。本方尽力确保信息准确,但不对信息的完整性、
准确性和时效性作出任何明示或暗示的保证。用户应自行核实相关信息。

二、服务限制
本【产品/服务】不提供【医疗/法律/投资/……】建议。本平台内容不构成专业意见,
用户在作出任何重要决策前,应咨询相关领域的专业人士。

三、责任限制
在法律允许的最大范围内,本方不对因使用本【产品/服务/内容】而产生的任何直接、
间接、偶发或后续损失承担责任。

四、第三方链接
本平台可能包含第三方网站或服务的链接,本方对第三方内容、隐私政策或行为不承担责任。

五、适用法律
本声明受中华人民共和国法律管辖,争议提交【公司所在地】人民法院解决。

六、更新
本声明可能不定期更新,更新后在本平台显著位置公告,继续使用即视为接受更新内容。

生效日期:【年】年【月】月【日】日
---
⚠️ 本草稿由 AI 辅助生成,仅供参考,正式使用前请经专业律师审核。
```

### 起草流程四步走

```
第一步:确认文书类型
  → 问清主人:要发给谁?要解决什么问题?期望对方做什么?

第二步:收集关键信息
  → 发函方信息(公司名/个人姓名/联系方式)
  → 收函方信息(对方名称/联系方式)
  → 事实描述(时间/地点/行为描述)
  → 法律依据(适用哪条规定)
  → 诉求(要对方做什么/期限是多久)

第三步:套模板填写
  → 按对应模板逐节填入,不得遗漏任何核心要素
  → 事实描述部分要具体、有可核查的细节

第四步:加标注交付
  → 文书顶部标"草稿 v1.0"
  → 用【方括号】标出需要主人确认或律师补充的空白项
  → 文末加免责声明
```

## 📝 毕业测验(必须真做,交证据)

**任务:起草两种不同类型的法律文书草稿。**

**指定组合(从下面选一组):**

- **组合 A:侵权告知函 + 版权声明**
  - 场景:主人公众号文章被某自媒体账号未经授权转载,需要发函要求删除并致歉;同时为公众号更新版权声明。

- **组合 B:免责声明 + 授权使用书**
  - 场景:主人的 AI 工具产品需要一份免责声明(AI 输出不构成专业建议);同时为一位合作方起草授权使用某套课程内容的授权书。

**你需要做到:**

1. **收集关键信息**:列出你对所选场景的信息假设清单(空白项用【待确认】标注)。

2. **产出两份完整草稿**:每份文书结构完整,核心要素齐全,事实描述具体,诉求清晰。

3. **每份文书**:标注"草稿 v1.0"、用方括号标出待确认项、文末含免责声明。

4. **自检**:对照"核心原则"五条,每条给自己的草稿打个分(满分 5 分),说明理由。

5. **沉淀技能卡**:把常见法律文书起草流程沉淀为 `skills/legal-docs.md`。

> ⚠️ **铁律:只产出草稿,绝不以主人名义发送任何函件、绝不联系任何第三方、绝不代表主人签署或提交任何文件。**
