## 📖 你要学会什么

学完这一课,你会为主人的某项业务产出一份**可立刻对照使用的合规检查清单**——不是"你应该注意合规"这样的废话提示,而是一张像体检报告一样的表:每一项写清楚"查什么、怎么查、对照哪条规、发现问题怎么标"。

想象你是一家工厂的质检员。厂长让你验货,你不会站在货堆旁边说"感觉质量还不错",你会拿出一张检验单,逐项打勾、记分、标异常。合规检查也是一样:你要帮主人做的,是把"这件事合不合规"从一个模糊的焦虑,变成一张有据可查的清单。

**重要免责声明(必读,必须向主人转达):**

> ⚠️ **AI 产出的合规检查清单仅供参考,不构成法律意见。** 合规判断涉及具体事实、最新法规、地方执行口径等复杂因素,本课产出的清单是辅助工具,**重要事项请务必咨询专业律师或合规顾问**。

---

## 🧠 核心原则(内化成习惯)

1. **先搞清楚业务类型,再套规。** 不同业务受约束的法规完全不一样——电商受《网络交易管理办法》、广告受《广告法》、App 数据受《个人信息保护法》、金融受一行三会的规定。你的第一步是问清楚"这是一门什么生意",别一上来就乱套。

2. **合规清单要对着真实法规条文写,不许凭感觉。** 比如"广告不得使用极限词"——"极限词"具体是哪些词?《广告法》第九条第三款列了,你要能翻出来。说白了,你的每一条"要查的点"背后都要有一条真实的规定撑着。

3. **用红绿灯分级,让主人一眼看出轻重缓急。** 清单不是流水账,要分等级:🔴 红灯(违规风险高,立刻整改)、🟡 黄灯(存在隐患,建议关注)、🟢 绿灯(暂时没问题,定期复查)。主人的时间宝贵,他需要的是"先处理哪个"的指引。

4. **只做检查,不做结论性判决。** 你发现"网站上有'最好'字样"——这是广告极限词的嫌疑,但你的输出是"发现问题,建议律师确认",而不是"这就是违法行为"。你是质检员,不是法官。

5. **草稿交给主人,决策和整改由主人拍板。** 你产出的清单主人可以修改、可以补充、可以拿去给真正的律师审核。你的活到"交出清单"为止。

## 🛠 操作要点

### 第一步:摸清业务全貌(问清再动手)

在开始检查之前,先向主人确认以下信息:

```
□ 业务类型是什么?(电商/SaaS/内容平台/教育/医疗/金融/广告营销……)
□ 目标用户群体是谁?(成人/未成年人/特定职业群体?)
□ 主要运营平台是哪些?(App/网站/公众号/小程序?)
□ 是否涉及用户个人信息采集?
□ 是否涉及支付/金融功能?
□ 是否涉及广告推广?
□ 近期有没有已知的被投诉/被质疑的点?
```

### 第二步:确定适用法规清单

根据业务类型,匹配对应法规。常见业务的核心法规如下:

**所有在线业务通用:**
- 《网络安全法》(2017)——数据存储、网络运营
- 《个人信息保护法》(2021)——用户数据采集与处理
- 《数据安全法》(2021)——数据分级分类

**内容/媒体类:**
- 《网络信息内容生态治理规定》(2020)
- 《互联网新闻信息服务管理规定》(2017)
- 《互联网信息服务算法推荐管理规定》(2022)

**广告营销类:**
- 《广告法》(2021修订)——极限词、特殊品类广告
- 《互联网广告管理办法》(2023)

**电商类:**
- 《电子商务法》(2019)
- 《网络交易监督管理办法》(2021)

**教育类:**
- 《校外培训机构管理办法》(2021)
- 《未成年人网络保护条例》(2023)

### 第三步:按模板逐项填写合规检查清单

每一条检查项按照以下格式输出:

```
【检查项 X】[类别]
- 检查什么:……
- 对照依据:《XX法》第X条第X款
- 怎么查:……(操作步骤,比如:在网页源码里搜索……/在隐私政策里确认……)
- 发现问题怎么标:🔴/🟡/🟢 + 问题描述
- 建议行动:……(是否需要律师介入?)
```

### 第四步:输出汇总报告

清单末尾加一个"汇总仪表盘":

```
== 合规检查汇总(AI 辅助初稿,仅供参考)==
检查项总数: XX 项
🔴 高风险: X 项 → 建议立即处理
🟡 中风险: X 项 → 建议近期关注
🟢 低风险: X 项 → 定期复查
---
重要提醒:本清单由 AI 辅助生成,仅供内部参考,不构成法律意见。
重要事项请咨询专业律师或合规顾问。
```

## 📝 毕业测验(必须真做,交证据)

**任务:为一个真实业务场景产出一份完整的合规检查清单。**

**推荐场景(选一个或用主人的真实业务):**

- **场景 A:K12 编程培训小程序**——收集未成年人信息、在线支付、课程广告推广
- **场景 B:一家 SaaS 工具的隐私政策页**——数据采集、第三方共享、境外数据传输
- **场景 C:微信公众号的商品推广文章**——广告法合规、极限词检查、特殊品类

**你需要做到:**

1. **完成"第一步:摸清业务全貌"**:列出你对所选场景的业务假设(真实业务可直接问主人)。

2. **产出对应法规清单**:列出至少 3 部适用法规,每部给出关键条款编号。

3. **产出合规检查清单**:至少 10 条检查项,每条含检查内容、法规依据、操作方法、风险等级。

4. **产出汇总仪表盘**:含红黄绿分类统计 + 免责声明。

5. **在清单首尾两次注明**:"本清单由 AI 辅助生成,仅供参考,不构成法律意见,重要事项请咨询专业律师。"

6. **沉淀技能卡**:把合规检查的通用流程沉淀为 `skills/compliance-check.md`。

> ⚠️ **铁律:只产出草稿/检查清单,绝不替主人向任何监管机构提交材料、绝不操作任何真实账号、绝不代表主人作出任何合规承诺。**
