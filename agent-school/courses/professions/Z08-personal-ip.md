## 📖 你要学会什么

学完这一课,你会帮主人产出一份**个人 IP 内容矩阵规划文件**——定位 + 选题库 + 栏目体系,三件事打包交付。这不是"写篇内容"这么简单,而是帮主人想清楚"以后往哪个方向持续产内容"。

先解释一下这件事的难度在哪里。

写一篇公众号或一条小红书,是"打一枪"。个人 IP 是"持续作战"——你需要在某个领域里,一年输出几十上百篇内容,还得越来越有辨识度。问题是:大多数人没想清楚自己的"根据地"在哪里,跟着热点走、什么火写什么,结果写了一年,读者还是不知道你是谁。

你的任务就是帮主人"挖护城河"——把他的经验、知识、独特视角整理出来,形成一套可持续产内容的系统,让关注他的人越来越清楚地知道"来找他可以得到什么"。

**知识付费这件事在国内的现状(真实调研,给主人看的):**

- **小报童**([xiaobot.net](https://xiaobot.net/)):国内个人创作者知识付费首选平台,支持专栏订阅、单篇付费。门槛低,无需公司主体。
- **知识星球**([zsxq.com](https://wx.zsxq.com/)):圈子制社群平台,适合做付费社群+内容沉淀。
- **得到 APP**([dedao.cn](https://www.dedao.cn/)):正式课程发行渠道,门槛较高,需申请入驻。
- **微信付费内容**:公众号付费文章 + 视频号付费直播,依托微信生态,转化路径短。

> ⚠️ 上述平台调研归调研,**登录任何平台账号、提交入驻申请、发布任何付费内容——全部须主人亲自操作**,你的活是产出规划文件。

## 🧠 核心原则(内化成习惯)

**1. IP 定位的"一刀切"测试**

最好的个人 IP 定位,是能用一句话说清楚的:"他是专门帮 ______ 解决 ______ 问题的人。"

填不进去的定位,要么太宽(什么都说),要么太空(说了等于没说)。帮主人填进这个句子之前,别开始写内容。

**2. 内容矩阵不是"多平台发布",而是"主阵地 + 分发网"**

想象一个渔网:中间有几个大节点(主要发力的平台),辐射出去是细线(其他平台用来引流)。主阵地只有一两个,其他都是放大器。让主人把 80% 的精力放在主阵地,内容矩阵才跑得起来。

**3. 选题库是"囤货",不是"现做现卖"**

很多创作者的崩溃点是"下期写什么"——大脑空白、拖延、一期烂尾。解决办法是提前建一个**选题库**:持续往里存话题、存灵感、存读者问题。你的任务就是帮主人搭这个仓库的架子,并第一批填进去至少 20 个选题。

**4. 栏目是"打标签让人记住你"的系统**

如果只发零散内容,读者记不住你;如果内容分栏目——"周一案例拆解""每月工具测评""不定期的那啥"——读者就有了期待,有了等你的理由。好的栏目有三个特征:可持续(主人能一直产)、有辨识度(只有他能做)、读者有期待感。

**5. 知识付费是"信任变现",不是"内容收费"**

人们为什么付费?不是为了内容本身(免费的太多了),是为了"这个人我信任,他整理好的比我自己找省事"。所以付费产品设计要围绕"帮他省什么"来想——省时间、省决策、省踩坑成本。

## 🛠 操作要点

### 第一步:梳理主人的"资本盘点"

在做任何规划之前,先把主人已有的东西摸清楚:

```
【经验盘点】
- 主人在什么领域有超过3年以上积累?
- 有没有外人眼中"稀缺"的经历或视角?
- 有没有已经在某个平台积累了粉丝/读者?

【内容存量】
- 仓库 content/posts/ 里有多少已发布内容?
- 这些内容的核心主题是什么?
- 已有哪些内容获得过较好的反馈?

【竞争差异】
- 同一领域,其他创作者在说什么?
- 主人有什么是别人没有的(经历/角度/用户群)?
```

### 第二步:产出 IP 定位文档

```
【IP 定位一句话】
他是专门帮 [目标人群] 解决 [核心问题] 的 [身份标签]。
示例:"专门帮想做内容的普通人,用 AI 把一个人的内容效率提升到以前三个人的水平"

【核心价值主张】
来关注他的人,能持续得到什么?(3条,每条一句话)

【人设三角】
- 专业标签:在 XX 领域有 XX 年经验
- 个性标签:说话风格是 XX
- 差异化:和其他同类创作者的最大不同是 XX

【主阵地选择】
- 主阵地(全力投入,2个以内):推荐 XX,理由是 XX
- 分发渠道(内容同步,不额外定制):XX、XX
- 暂不投入的平台:XX(原因:XX)
```

### 第三步:搭建栏目体系

```
【固定栏目(2-4个)】
每个栏目说清楚:
- 栏目名
- 更新频率
- 每期产出什么
- 为什么只有主人做得出来

示例:
┌─────────────────────────────────────────────────┐
│ 栏目:AI 工具真实测评                            │
│ 更新:每月 2 次                                  │
│ 产出:1000 字测评 + 使用截图 + 推不推荐结论       │
│ 差异:我真的花时间用,不是看说明书写的            │
└─────────────────────────────────────────────────┘

【不定期内容(1-2类)】
- 话题:有感而发 / 行业大事评论
- 协作:联合其他创作者的内容
```

### 第四步:建立选题库(首批至少 20 条)

选题库的分类方式:

```
【A类:长青选题】(搜索量稳定,长期有效)
- 新手常见误区系列
- 工具使用教程系列
- 经典案例拆解系列

【B类:热点选题】(借势当前流量)
- 行业新闻评论
- 平台规则变化解读
- 新工具首发测评

【C类:深度选题】(建立专业壁垒)
- 系统化方法论
- 行业趋势判断
- 对比研究
```

每条选题要写:标题想法 + 核心观点一句话 + 适合哪个栏目/平台 + 难度估计

### 第五步:知识付费产品设计建议(草案)

```
【入门级产品(0-99元)】
- 形态:付费专栏 / 资料包
- 内容:主人最实用的那套方法/模板/清单
- 平台:小报童 / 公众号付费文章

【中间层产品(199-699元)】
- 形态:训练营 / 课程包
- 内容:从零到一解决某个具体问题
- 平台:知识星球 / 微信群 + 视频号

【高价值产品(1000元以上)】
- 形态:私教/陪跑/咨询
- 内容:1对1或小组深度服务
- 转化路径:从免费内容→入门产品→高客单价
```

> ⚠️ 以上产品设计是草案建议,**定价、发布、开课——全部须主人亲自决策和操作**。你的活是给主人一个可参考的框架,不是替他敲定并执行。

## 🧰 配套开源项目(可选集成)

- **humanizer** ([github.com/blader/humanizer](https://github.com/blader/humanizer), 活跃项目) —— 去 AI 腔工具,让批量产出的 IP 内容读起来更像"真人说的"。**用法**:内容矩阵批量生产后,把各平台版本逐一过 humanizer 去除模板腔,再提交给主人审核。个人 IP 的"人味"是核心竞争力,这一步不建议省。国内可用,安装须主人确认。
- **marketingskills(customer-research + marketing-psychology)** ([github.com/coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills), ~33k ⭐) —— 营销全家桶,其中 customer-research 和 marketing-psychology 模块尤其适合本课:帮主人提炼目标人群画像、找准 IP 定位的心理锚点。**用法**:在梳理"资本盘点"和"IP 定位"阶段,调用 customer-research Skill 生成用户画像草稿。⚠️ **该项目以英文内容为主,中文场景需要额外提示词本地化调整**。

## 📝 毕业测验(必须真做,交证据)

**任务:为主人(或一个假设的创作者)产出完整的个人 IP 内容矩阵规划包。**

**设定(选一个或结合实际):**
- 假设主人是一个用 AI 做内容的自由职业者,想把自己的方法论做成付费专栏
- 假设主人是某垂直领域(电商/教育/设计……)的从业者,想开始做个人 IP

**你需要做到:**

1. **资本盘点表**:先去 `content/posts/` 翻已有内容,列出:现有内容主题分布、已有优势方向、空白或可补强的方向。(没有实际内容的,基于假设填写)

2. **IP 定位文档**:完整填写上方的"IP 定位一句话 + 核心价值主张 + 人设三角 + 主阵地选择"。

3. **栏目体系**:设计 2-3 个固定栏目 + 1 类不定期内容,每个栏目写清楚名称/频率/产出/差异化。

4. **选题库首批**:在选题库里填入 **20 条选题**(A/B/C 三类各有覆盖),每条含:标题想法 + 核心观点 + 适合平台 + 难度。

5. **知识付费产品草案**:设计 2 个层级的产品(不需要3层),写清楚:定位、内容、推荐平台、合理定价范围建议。

6. **沉淀技能卡**:把个人 IP 内容矩阵规划流程沉淀成 `skills/personal-ip-matrix.md`。

> ⚠️ **铁律再重申:只交规划草稿,绝不替主人注册账号、提交入驻申请、发布任何付费内容。** 你的活到"把规划文件写好交给主人"为止。决策和执行是主人的权力。
