## 📖 你要学会什么

学完这一课,你会帮主人产出一份**可以立刻送去上架的电商详情页内容包**——卖点提炼 + 详情页文案结构 + 各平台适配说明,三件事打包交付。

先说说电商详情页这件事的"生死逻辑"。

一个消费者刷到你的商品,他做的决策不是"这个好不好",而是"我要不要相信这个产品能帮我"。这个相信,是在短短几秒到几十秒的滑动里建立起来的。

所以电商详情页的本质是**一个说服流程**:从"我在看什么"到"我为什么要买"到"我现在就买"。每一屏内容都有任务,没有废屏。

你需要理解三个平台的不同"物理法则":

**淘宝/天猫**:用户坐在家里、手机或电脑、有充足的时间比较——详情页可以比较长,支持图文混排,买家可以看很久再决策。重视评价、旗舰店权威感。

**拼多多**:用户主要看价格和口碑——详情页要极度简洁,核心卖点要大字显示,价格要放在最显眼的位置。不要啰嗦,买的就是快和便宜。

**抖音电商**:用户是在刷视频时被"突然打动"的——详情页配合视频脚本,主视觉冲击力要强,文字少、图片大、第一屏必须锁定注意力。

**关于现成工具(先调研,装前必须问主人):**

- **humanizer**:电商文案最容易有 AI 腔(读起来像机器人在夸奖),写完必须过一遍去 AI 腔。链接:[github.com/blader/humanizer](https://github.com/blader/humanizer)

> ⚠️ 本课重点是产出文案草稿,绝不替主人登录店铺后台、上架商品、发布任何内容。主人有没有店铺账号、想不想上架——全是主人的事。

## 🧠 核心原则(内化成习惯)

**1. 先挖痛点,再讲卖点**

新手写详情页犯的最大错误:一上来就说"本产品采用 XX 材质,具备 XX 功能"。

读者看到这里会想:跟我有什么关系?

正确顺序是:先让买家感受到"没有这个东西的痛"——"每次出门找不到钥匙……""熬夜写稿第二天黑眼圈……"——然后你的产品像救星一样出现。

先刺痛,再解痛。

**2. 卖点 ≠ 功能,卖点 = 功能带来的好处**

功能是:"采用 ANC 主动降噪技术"
卖点是:"地铁上也能安静工作,再也不被噪音打断思路"

一个测试:在每个功能后面问"所以呢?这对买家意味着什么?"——能清晰回答的才是卖点。

**3. 每一屏都有任务,没有废屏**

想象买家从上往下滑,每一屏解决一个问题:
- 第一屏:抓眼球(我在看什么?值得继续看吗?)
- 第二屏:建立渴望(我为什么需要这个?)
- 第三屏起:消除顾虑(质量可靠吗?划算吗?和其他的比怎样?)
- 最后一屏:促成行动(现在就买/加购)

**4. 不同平台用不同"语气"**

淘宝:稳重、详细、可以长一点
拼多多:直白、有力、突出低价和高性价比
抖音电商:冲击、感情牌、视觉先行

**5. 草稿是草稿,发布是发布**

全程不要登录任何电商后台、不要提交商品。你的活到"把文案包写好交给主人"为止。

## 🛠 操作要点

### 第一步:卖点提炼工作表

在动笔写之前,先和主人把这张表填完:

```
【产品基础信息】
品类:_____
定价:_____
目标人群:谁会买?(年龄/性别/使用场景)

【痛点挖掘】
目标买家现在最头疼的是什么?
没有这个产品,他们是怎么凑合的?(凑合方案的缺点是?)

【核心卖点(最多5条,优先级排序)】
1. _____ → 对买家意味着:_____
2. _____ → 对买家意味着:_____
3. _____ → 对买家意味着:_____
(可选)4. _____
(可选)5. _____

【竞品对比亮点】
和同类竞品相比,主人的产品哪里更好?哪里不同?
(如果没有明显优势,坦率告知主人,别硬编)

【真实用户反馈】(如果有)
买家最常夸的是什么?
买家问得最多的问题是什么?(这些要在详情页里提前解答)
```

### 第二步:详情页结构(通用版 + 各平台差异)

**通用结构(淘宝/天猫标准版)**

```
[主图区] 主视觉 + 核心卖点一句话(不超过15字)

屏 1:场景钩子
  → 目标买家的真实使用场景或痛点(图+少量文字)
  → 让买家觉得"这说的就是我"

屏 2:产品亮相 + 核心卖点
  → 产品主图(不同角度)
  → 核心卖点 3-5 条,每条一句话+一张局部图

屏 3:功能/材质细节
  → 每个功能细节配图+文字说明
  → 格式:"【功能名】→ 对你意味着……"

屏 4:对比图(可选但强烈推荐)
  → 用前用后 / 有无对比 / 和竞品对比
  → 让买家"看见"差异,不只是"听说"

屏 5:买家真实评价截图(如有)
  → 选最有说服力的,图文结合

屏 6:使用说明 / FAQ
  → 提前回答买家最常问的 3-5 个问题
  → 降低退货率

屏 7:收尾 + 行动号召
  → 品牌/店铺信任背书
  → "现在下单……"/"限时……"
```

**拼多多简化版**

```
删掉背景铺垫,开门见山:
屏 1:大字价格 + 核心卖点一句话
屏 2:产品图(多角度)+ 规格参数
屏 3:真实买家评价(越多越好)
屏 4:发货/售后保障说明
```

**抖音电商版**

```
第一屏:强冲击主视觉(大图/颜色对比/前后对比)
文字压到最少,让画面说话
全程用短句,像在说话:"就这一款""真的绝了""这个真不坑"
```

### 文案风格参考

| 平台 | 推荐文案风格 | 示例 |
|------|------------|------|
| 淘宝 | 详细、专业、有温度 | "精选 XX 级材料,经过 48 道工序……每一步都是为了让你用得放心" |
| 拼多多 | 直接、口语、强调划算 | "同款工厂货,别家卖 99,我这里只要 39!不服来比" |
| 抖音 | 情绪化、场景感、短促 | "这个我真的用了才知道有多好用……" |

### 长文案 vs 短文案选择

```
产品客单价 < 100元:简洁版(3-4 屏够了,买家不想看太多)
产品客单价 100-500元:标准版(5-7 屏,需要建立足够信任)
产品客单价 > 500元:详细版(8 屏以上,要彻底消除顾虑)
```

## 📝 毕业测验(必须真做,交证据)

**任务:为一款真实或假设的产品,产出完整的电商详情页内容包。**

**推荐产品(选一个或自定):**
- AI 工具/软件会员服务(适合抖音电商)
- 家用小电器(适合淘宝)
- 食品/零食(适合拼多多)
- 学习类课程产品(适合淘宝/抖音电商)

**你需要做到:**

1. **卖点提炼工作表**:完整填写上方工作表(产品信息 + 痛点挖掘 + 5条卖点 + 功能→好处转化 + 竞品亮点)。

2. **淘宝版完整详情页文案**:按通用结构写出 7 屏的文案脚本,每屏写明:这屏的任务 + 文字内容 + 配图方向(不用真的做图,写出"这里放一张 XX 的图"即可)。

3. **拼多多适配版**:基于淘宝版内容,简化成 4 屏的拼多多版,写明改了什么、为什么改。

4. **抖音电商适配版**:写出第一屏的主视觉文案(20 字以内)+全篇的短句版文字(每句不超过 15 字)。

5. **自检**:对照以下两条检查——(1)每条卖点后面有没有"对买家意味着什么"的解释?(2)三个平台版本的语气是否有明显区别?

6. **沉淀技能卡**:把电商详情页内容包流程沉淀成 `skills/ecommerce-detail.md`。

> ⚠️ **铁律再重申:只交草稿,绝不上架。** 全程不要登录任何淘宝/拼多多/抖音电商后台、不要提交商品、不要发布内容。你的活到"把文案包写好交给主人"为止。
