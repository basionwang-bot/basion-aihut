## 📖 你要学会什么

学完这一课，你会产出一批**可直接交给投放团队的广告文案变体包**——不是一条广告语，而是同一个卖点的 10~20 条变体，针对不同角度、不同受众、不同情绪，一次性让投放同学有东西可测。

想象你是一个**广告文案的"克隆工厂"**。投放这件事的残酷真相是：一条素材跑一段时间就"跑烂了"——用户见过了，点击率下降。投放同学每隔几天就需要新素材。你的工作不是产出"一条完美文案"，而是产出**一批可测试的变体**，让数据来告诉主人哪条最好使。

**本课产出的是文案文字部分，图片/视频由主人的设计团队制作，你只负责文字草稿。**

## 🧠 核心原则

### 1. 信息流广告的黄金 3 秒

无论是抖音开屏广告、快手信息流还是微博信息流，用户在内容流里刷到你的广告只有 **3 秒决定要不要停下来**。所以文案最重要的不是"说了什么"，而是"第一句话能不能让人停"。

好的第一句话有三种起手式：
- **扎痛点**："你还在用那种洗完头发两小时就出油的洗发水吗？"
- **给惊喜**："这个功能我用了 3 年才发现，一直以为是没有的……"
- **反常识**："月薪 5000 的人，为什么可以住得比 3 万的人更好？"

### 2. 变体不是改几个字，而是换"视角"

真正的变体是从不同维度切入同一个产品：
- 换**痛点角度**（失眠 vs. 皮肤差 vs. 工作效率低）
- 换**人群角度**（宝妈 vs. 职场人 vs. 学生）
- 换**情绪调性**（紧张焦虑型 vs. 温暖励志型 vs. 幽默轻松型）
- 换**信任来源**（用户口碑 vs. 专家背书 vs. 数据/成分）

### 3. 巨量和磁力的风格有差别

- **巨量引擎（抖音/今日头条）**：用户偏年轻，接受感官刺激、节奏快的文案；"爆款词"迭代快，要追热词
- **磁力引擎（快手）**：用户更偏下沉、家庭导向；"接地气""真实感"比"高大上"更管用；主播/达人种草文案效果好

同一批素材，有时候要针对两个平台出两个版本。

### 4. 合规红线

- 不能用"最""第一""唯一""100%有效"等极限词（违反广告法）
- 医疗/健康类产品不能有"治愈""根治"等效果承诺
- 金融类产品不能保证收益
- 广告标注"广告"二字是平台硬要求，你不需要写进文案，但要知道这一点

## 🛠 操作要点

### 广告文案变体包的标准结构

```
投放文案变体包（交给投放团队直接用）：
├── 产品信息头          主人填写（产品名/卖点/目标人群/平台）
├── 主标题变体 ×10      不同角度的第一句话
├── 描述文案变体 ×10    与主标题搭配的展开说明（每条50~80字）
├── CTA（行动号召）×5   引导用户点击/购买的结尾文案
└── 创意方向说明        给设计同学的配图/配视频建议
```

### 变体生产工作流

**第一步：填写产品信息摘要**
```
产品：___________
核心卖点（1~3个）：___________
目标人群（写具体，不是"所有人"）：___________
投放平台：巨量 / 磁力 / 两者都要
情绪基调：紧张焦虑 / 温暖励志 / 幽默轻松 / 理性专业（可多选）
预算级别：测试期（控制变体数量）/ 放量期（多多益善）
```

**第二步：按"视角矩阵"展开**

| 视角维度 | 示例切入 |
|----------|---------|
| 痛点A × 人群a | 宝妈担心孩子睡眠不好 |
| 痛点A × 人群b | 职场人担心自己睡眠不好 |
| 痛点B × 人群a | 宝妈担心家里空气质量 |
| 惊喜钩子 × 人群a | 有个小方法宝妈可能没听过 |
| 数据背书 × 人群b | XX万职场人都在用的方法 |

把矩阵格子填出来，每个格子产出 1~2 条文案，凑够目标数量。

### 真实示例：国产空气净化器（巨量引擎投放）

**产品信息**
- 产品：家用空气净化器，适用 30m² 房间
- 核心卖点：HEPA 滤网，去除 PM2.5 / 甲醛 / 宠物毛发
- 目标人群：有娃家庭（0~6岁）、新装修入住、养猫狗人群
- 平台：巨量引擎（抖音）

---

**主标题变体 × 10（第一句话）**

| # | 视角 | 文案 |
|---|------|------|
| 1 | 宝妈·焦虑型 | 孩子一直咳，换了好几次药，后来才知道是家里空气的问题 |
| 2 | 宝妈·数据型 | 国内室内PM2.5超标是室外的5倍，你家娃每天呼吸的是什么？ |
| 3 | 养宠·痛点型 | 家里有猫的都懂：衣服一出门就是满身毛，根本扫不完 |
| 4 | 养宠·轻松型 | 终于有产品让我不用每天跟猫毛打仗了…… |
| 5 | 新装修·焦虑型 | 新房装修完才搬进去，孩子眼睛一直红，你知道是什么原因吗？ |
| 6 | 新装修·惊喜型 | 甲醛超标没有味道——这是很多人不知道的事 |
| 7 | 通用·反常识型 | 关了窗反而更脏？室内空气污染你可能理解反了 |
| 8 | 通用·口碑型 | "买之前没想到，现在不开它我会不放心"——用户说的 |
| 9 | 通用·成分背书型 | 同款HEPA技术，医院手术室也在用，现在进了家庭 |
| 10 | 通用·温暖型 | 给家人的空气质量，和给家人用的食物一样重要 |

---

**描述文案变体 × 5（搭配第1条标题）**

| # | 描述（50~80字） |
|---|----------------|
| 1 | 去了医院才发现，家里PM2.5常年超标。换了净化器第二周，孩子夜里咳嗽明显少了。不是所有问题都出在外面——室内空气也是。双层HEPA滤网，覆盖PM2.5 / 甲醛 / 过敏原，静音运转，孩子睡觉都不会吵醒。 |
| 2 | 我试过好几款，贵的噪音大，便宜的三个月滤芯就堵死了。这款换滤芯的提示很准，而且滤芯不贵，用起来没有隐形消耗的焦虑。有娃家庭真心推荐。 |
| 3 | 30平米房间，2小时完成一次净化循环。晚上孩子睡觉开着，早上起来能看到收集到的灰尘量——每次看都心惊，这些要是吸进去的话。 |
| 4 | 养宠+有娃双重场景，同时处理宠物毛发和PM2.5，过滤精度达到0.3微米。开机3小时，摘下面罩能感觉到空气不一样了。 |
| 5 | 新房入住前后各测了一次甲醛——进来之前2.5倍超标，净化器跑了1个月之后降到正常范围。感觉终于能放心住进去了。 |

---

**CTA 变体 × 5**

```
1. 点击了解型：→ 了解详细参数，看哪款适合你家面积
2. 紧迫感型：→ 限时补贴中，点进去看看还有没有
3. 低风险型：→ 先看评价和检测报告，再决定要不要买
4. 社交型：→ 评论区告诉我，你家现在用什么净化器？
5. 问题引导型：→ 不知道选哪款？主页有选购指南免费看
```

**创意方向（给设计同学）**

```
主力方向：真实室内场景（有孩子的客厅/卧室）
- 图1：孩子在地上玩 + 净化器静静运转，温馨感
- 图2：滤芯对比（旧vs新），视觉冲击型
- 图3：PM2.5检测仪数值变化，数据信任型
视频：15秒，前3秒用灰尘/宠物毛/甲醛试剂条画面钩住，后12秒展示产品运转+数值变化
```

## 📝 毕业测验（必须真做，交证据）

**任务：为一款真实或假设产品，产出完整投放文案变体包。**

**推荐场景（选一个）：**
- 国产祛痘护肤品，巨量引擎，面向 18~28 岁女性
- 国产便携蓝牙音箱，磁力引擎，面向 25~40 岁户外爱好者
- 在线英语课，同时投巨量+磁力，面向宝妈

**你需要做到：**

1. **产出变体包全套**：主标题 ×10 + 描述文案 ×10 + CTA ×5 + 创意方向说明。

2. **视角矩阵说明**：写出你的"视角矩阵"——至少 3 个痛点维度 × 2 个人群切入，共覆盖 6 个组合，说明每组的逻辑。

3. **平台适配**（如选了双平台）：说明巨量版本和磁力版本有哪些具体差异，至少给出 2 条差异化改写对比。

4. **合规自检**：把 10 条主标题过一遍，列出"已避开的极限词"（至少 2 条），说明改法。

5. **沉淀技能卡**：把投放文案变体生产流程沉淀成 `skills/ad-creative.md`。

> ⚠️ **铁律：只产文案草稿，绝不投放。** 全程不登录任何巨量引擎/磁力引擎广告账户、不创建任何广告计划、不上传任何素材到真实平台。你的活到"把变体包文件写好交给主人"为止。
