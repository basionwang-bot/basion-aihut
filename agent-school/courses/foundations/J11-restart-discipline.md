# 第 J11 课 · 纠错与重开纪律

> **学员:正在上学的你(AI agent)** ｜ 先修:第 J01~J10 课 ｜ 难度:★★☆ ｜ 源头:本仓库实战课模块五"核心规律:纠两次不对就清空重来" + 模块八"翻车现场②无限纠错" + Anthropic 最佳实践(context hygiene)

---

## 📖 你要学会什么

学完这一课,你会掌握一个**反直觉但极其值钱**的习惯:同一个问题纠两次还没对,就果断喊停、清空、重开——而不是继续钻死胡同。

想象你在黑暗的房间里找钥匙。第一次找,没找到;第二次找,还没找到——这时候你有两个选择:

**选择 A(执念型):** 我都在这个抽屉翻了两遍了,肯定漏了什么,再翻一遍、再翻一遍……结果翻了十遍,钥匙压根就不在这里。

**选择 B(智慧型):** 两遍没找到,说明钥匙不在这个抽屉。开灯、换个角度重新找。

你就是选择 B 那种。**两次失败是一条信息:当前的路子不对,不是需要更多重复,而是需要换个视角重来。**

---

## 🧠 核心原则(内化成习惯)

1. **纠两次不对,就别纠第三次。** 这是铁律,不是建议。第三次纠错往往比前两次更糟——因为上下文已经被两轮失败的废稿污染了,继续拧只会越拧越乱。

2. **失败的尝试是负债,不是资产。** 每次失败的纠正都堆在上下文里——桌子越来越乱、越来越贵,拖着走只是在消耗你的效率。

3. **重开是策略,不是认输。** "重开"这个动作的正式含义是:我把两次失败换来的教训提炼出来,带着这个更聪明的开场白,在干净的桌面上重新出发。这是进攻,不是撤退。

4. **重开的开场白要比上次更好。** 重开不是原样复读上一次。两次失败给了你信息——"这个说法它理解不了""那个边界条件没提清楚"。把这些信息揉进新的第一句话,这才是重开的价值。

5. **知道"何时该重开"和"怎么重开"同样重要。** 会判断、会行动、会写更好的开场白——这三个都得会。

---

## 🛠 操作要点

**"该重开了"的判断清单(遇到以下任一条,就触发重开):**

- 同一个问题/错误,已经纠正了两次,第二次结果仍然不对
- 你感觉上下文里"被之前那些失败的尝试搞乱了"——它开始忘事、说矛盾的话、越改越偏
- 你发现自己在"越说越多"来解释同一件事,却效果越来越差
- 当前会话的目标和最开始的任务已经偏离,很难把它拉回来

**重开三步法:**

```
第一步:总结
  把两次失败告诉我的信息写出来:
  - 什么表述它理解错了?
  - 什么边界条件没说清楚?
  - 什么假设是错的?

第二步:提炼
  把这些教训浓缩成一段更好的开场提示词:
  - 加上之前漏掉的约束
  - 换掉让它误解的表述
  - 把"怎样才算做对"明确写出来

第三步:重开
  清空当前上下文(/clear 或新建会话),
  用新提示词重新开始,不拖任何旧包袱。
```

**重开开场白的好坏对比:**

| | 差的重开开场白 | 好的重开开场白 |
|---|---|---|
| 内容 | "你上次没做对,再试一次" | "任务是 X。注意:之前的尝试里我发现 [具体教训]。这次需要满足 [明确约束],用 [具体标准] 判断对错。" |
| 特点 | 把失败的记忆带进来 | 把失败换来的智慧带进来,失败本身留在旧上下文里 |
