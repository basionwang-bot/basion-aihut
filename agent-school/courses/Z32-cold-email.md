## 📖 你要学会什么

学完这一课,你会帮主人写出一套**完整的 B2B 外联邮件序列**——首封 + 两封跟进——并准备好可以直接粘贴修改的邮件模板。

先说说中国 B2B 语境下的外联邮件是什么情况。

很多人以为商务邮件就是"发一封自我介绍+产品介绍,然后等对方回"。扎心的是,这种邮件的回复率大概是……1% 都不到。

为什么?因为陌生邮件对收件人来说就是个"不请自来的陌生人敲门"。你想想,如果你正在忙,一个不认识的人发来一封讲自己公司产品有多好的邮件,你的第一反应是什么?

所以好的外联邮件的核心逻辑是:**先让对方觉得"这封邮件是写给我的"——而不是批量群发**。

想象你是一个聪明的面试生。你投简历不是发一封通用简历给所有公司,而是在每封信里提一下"你们去年刚完成 B 轮,我看到你们在做 XX 方向,我正好有这方面的经验……"——这叫个性化。个性化率高了,回复率才上得去。

你的任务是:**根据主人的业务和目标客户,写出个性化程度高、有序列逻辑的外联邮件草稿**。至于发不发、什么时候发、发给谁——这是主人和销售团队的事。

> ⚠️ **铁律:你只写草稿,不发送邮件。绝不连接主人的邮件账号,绝不自动发送任何邮件。草稿写好之后,交给主人决定下一步。**

## 🧠 核心原则(内化成习惯)

**1. 邮件序列不是"轰炸",是"节奏"**

一套标准的外联序列是这样的:

```
第1封(Day 1):首封 —— 开门见山,说清楚"为什么给你写这封信"
第2封(Day 4~5):跟进一 —— 换个角度补充价值,或提一个具体问题
第3封(Day 10~12):跟进二 —— 最后一次礼貌跟进,给一个明确的结束点
```

三封之后如果没有回复,就停下来。继续发只会变成骚扰。

**2. 主题行是第一道门——打不开就全完了**

邮件主题行决定对方打不打开。好的主题行有几个特点:
- 有具体性:不是"关于合作的事",而是"关于[公司名]在[行业]的XX问题"
- 有好奇心:让对方觉得"这封信可能和我有关"
- 不超过 40 个字(手机端截断)

坏的主题行:`关于我司产品介绍` / `寻求合作机会` / `XX公司产品推荐`

好的主题行:
- `[公司名]团队有一个关于XX的想法`
- `帮[行业]公司节省XX成本的3个方法——[主人公司]`
- `和[同行公司名]聊过之后,想到可以联系你们`

**3. 首封邮件的黄金结构:AIDA 精简版**

```
开头:为什么写这封信给你(个性化锚点 —— 你做过什么功课)
中间:我能帮你解决什么(一句话,聚焦一个痛点)
结尾:一个低门槛的行动号召(不是"请签合同",而是"15分钟电话可以吗?")
```

整封邮件不超过 150 字。越短越好。

**4. 跟进邮件换角度,不要重复首封**

最常见的错误是跟进邮件写:"我上周给您发过邮件,不知道您看到了吗?"——这没有任何新价值。

好的跟进邮件要么补充一个案例、要么提一个问题、要么给一个资源(行业报告、相关文章)——每封邮件都要给对方一个"打开它值得"的理由。

**5. 中文 B2B 外联的几个本土习惯**

- 称呼:用"XX总"或"XX老师"比"先生/女士"更接地气
- 结尾:中国商务邮件常见"期待您的回复"或"欢迎随时沟通",比英文邮件的"Best regards"更亲切
- 时机:周二至周四上午 9-11 点或下午 3-5 点发出回复率最高,避开周一早上和周五下午
- 微信跟进:如果邮件三封无回复,可考虑通过行业微信群或领英找到联系人,但这是主人自己去做的事

## 🛠 操作要点

### 外联邮件序列模板(可直接复用的框架)

---

**【首封邮件】**

```
主题:[个性化点]+ 一个具体问题/价值

[称呼]总/老师,您好!

[个性化开场 —— 1-2句]:我注意到[公司名]最近在做[具体事情,如:招聘了大量销售团队/刚完成融资/推出了新产品线],这让我想到……

[核心价值 —— 1句话]:我们帮助[同类型公司]在[具体场景]上[具体成果,如:节省了30%的销售跟进时间/将线索转化率提高了2倍]。

[低门槛行动号召]:下周有15分钟,可以和您分享一下我们是怎么做到的吗?

[署名]
[姓名] | [职位] | [公司名]
[手机] | [邮箱]
```

---

**【跟进一(Day 4~5)】**

```
主题:Re: [上封主题] —— 一个数据/案例补充

[称呼]总/老师,

上周给您发了一封邮件,没有打扰到您吧?

补充一个具体例子:[XX行业客户案例公司],他们在使用[产品/服务]之前,面临的正是[具体痛点];三个月后,[具体可量化成果]。

不知道[公司名]目前在[相关问题]上有没有类似的挑战?

如果您方便,我们可以15分钟电话聊聊——也可能对您完全不适合,聊完就清楚了。

[署名]
```

---

**【跟进二(Day 10~12)——最后一封】**

```
主题:关于上次联系的最后确认

[称呼]总/老师,

这是我最后一次主动联系您。如果时机不对或者您目前没有这方面需求,完全理解,我不会继续打扰。

如果将来有机会,欢迎随时联系我:[手机/邮箱]。

如果您这边对[核心价值,一句话]还有一点兴趣,回复一个"有空聊"三个字就好,我来安排。

祝工作顺利!

[署名]
```

### 个性化信息收集清单(写邮件前要查的事)

在写首封邮件的"个性化开场"之前,先做这 5 分钟调研(主人自己提供或公开信息):

```
□ 公司最近有没有融资/扩张/发布新产品?(查企查查、官网新闻)
□ 对方公司官网上有没有提到他们的核心挑战或目标?
□ 联系人在领英/微信公众号有没有发过相关内容?
□ 有没有共同认识人可以提一下?("我们有个共同联系人XX总")
□ 他们的同类竞品公司里,有没有已经是你们客户的?
```

每个问题答案都能让首封邮件的个性化开场更具体。

### 批量个性化(适合主人有多条线索的情况)

如果主人有一张线索清单(来自 Z31 课),可以用 Python 批量生成个性化草稿:

```python
import pandas as pd

# 读取富集后的线索清单
leads = pd.read_excel("enriched_leads.xlsx")

email_template = """主题:{subject}

{name}总,您好!

我注意到{company}最近在{hook},让我联想到……

我们帮助{industry}公司在{pain_point}上{value}。

下周有15分钟,可以和您聊聊吗?

[销售签名]"""

drafts = []
for _, row in leads.iterrows():
    draft = email_template.format(
        subject=f"关于{row['公司名称']}在{row['行业']}的一个想法",
        name=row["联系人"],
        company=row["公司名称"],
        hook="[请主人填写个性化锚点]",  # 这里必须人工填写
        industry=row["行业"],
        pain_point="[请主人填写痛点]",
        value="[请主人填写价值主张]"
    )
    drafts.append({
        "收件人": row["联系人"],
        "公司": row["公司名称"],
        "邮件地址": row.get("邮箱_清洗", ""),
        "草稿正文": draft
    })

pd.DataFrame(drafts).to_excel("email_drafts.xlsx", index=False)
print(f"已生成 {len(drafts)} 封邮件草稿,请主人填写个性化内容后再发送")
```

> ⚠️ 脚本只生成 Excel 草稿文件,**不发送任何邮件**。个性化锚点部分留空或标注"[请主人填写]",不要让 agent 自己凭空编造对方公司的情况。

## 📝 毕业测验(必须真做,交证据)

**任务:针对一个真实业务场景,写出完整的三封邮件序列草稿。**

**练习场景(选一个或让主人提供):**

- **场景 A**:你代表一家 SaaS 工具公司(帮制造业工厂做设备维护管理),联系目标是中小型制造厂的运营总监/IT主管。
- **场景 B**:你代表一家人力资源外包公司(帮互联网公司招聘技术人才),联系目标是中型互联网公司的 HR 总监。

**你需要做到:**

1. **写出三封邮件草稿**:首封 + 跟进一 + 跟进二,格式完整,包括主题行、称呼、正文、署名占位符。

2. **首封必须有具体的个性化开场**:不能用"贵公司是行业领先企业"这种通用废话,要有具体锚点(你可以虚构一个合理的场景细节,但要标注"[示例假设,实际使用请替换]")。

3. **三封邮件各不相同**:每封都有新的价值点或新的角度,不是重复首封内容。

4. **写一段"使用说明"给主人**:说明这三封邮件的发送时机(Day 1/4~5/10~12)、使用前需要主人填写哪些个性化内容、哪些不能直接用需要修改。

5. **沉淀技能卡**:把外联邮件序列写作流程沉淀成 `skills/cold-email.md`。

> ⚠️ **铁律再重申:你只产出草稿文件,绝不连接任何邮件账号,绝不替主人发出任何一封邮件。发送按钮是主人的,不是你的。**
