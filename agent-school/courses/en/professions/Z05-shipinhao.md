## 📖 What you'll learn

After this lesson, you will deliver a **complete short-form vertical video content pack** — a text post / caption draft + a short video script + cross-platform distribution advice, bundled together.

First, an important reality check.

**There is no mature, dedicated AI tooling for Instagram Reels or YouTube Shorts.** Unlike newsletter platforms (Ghost, Substack, Mailchimp) or Instagram feed posts (Canva, Later), dedicated tools for short-form vertical video are a thin layer — mostly repurposed TikTok workflows or general video editors. This is the reality as of 2026, not something being hidden from you — knowing the real situation is more useful than a list of tools that don't exist.

But that doesn't stop you from producing great content. The core value of short-form vertical video isn't "how many AI tools help you make it" — it's the **distinct distribution mechanic of each platform**. Reels get discovered through the Instagram explore feed and shares; Shorts get distributed through YouTube's algorithm and subscribers. Understanding that logic tells you how to write the content.

## 🧠 Core principles (internalize these as habits)

**1. The "terrain": distribution is social + algorithmic, not purely algorithmic**

TikTok is pure algorithm push — if your content hooks strangers, the algorithm pushes it to more strangers.

Instagram Reels and YouTube Shorts blend algorithm push with **social graph amplification**: Reels can reach non-followers through Explore and through shares; Shorts surface in subscriptions and in YouTube's recommendation sidebar. Your content needs to work for both pathways — it must hook new eyes *and* be worth sharing to people who know the creator.

**2. Tone: warm, useful, unpretentious**

People share short-form vertical videos because they are one of three things:
- **Practically useful**: "This tip is so good, I have to send this to someone"
- **Resonant**: "This says exactly what I've been thinking"
- **Emotionally touching**: "I want my friend to see this"

**3. Caption vs. short video: different tools for different jobs**

Both Reels and Shorts allow companion text (captions, descriptions). You need to advise the owner on which format to lead with:
- **Caption / text post** (Instagram): lower production cost, good for quick publishing and knowledge shares; high shareability in Stories and DMs
- **Short video (Reels / Shorts)**: stronger distribution reach; requires the owner to actually film something

**4. Draft is draft, publish is publish**

Same iron rule as Z01: your job ends at "hand the content pack to the owner." Logging in and posting — those are the owner's moves, never yours.

## 🛠 How to do it

### Structure of a companion caption (Instagram Reels or YouTube Shorts description)

Think of the caption as **a punchy friend's note that accompanies the video**:

```
[Opening line]
The most important observation or most resonant line goes first
(on Instagram this is the only text visible before "more")

[Body]
3–5 paragraphs, 2–4 lines each
Include a concrete scenario or example
Conversational tone, genuine warmth

[Closing]
A one-line summary OR a question that invites connection
Add hashtags if Instagram: #topic (2–5 tags max, keep it clean)
YouTube descriptions: add timestamps and relevant links instead
```

### Short video script structure

Optimal length for Reels: 15–60 seconds. For Shorts: 15–60 seconds (up to 3 minutes but shorter performs better).

```
[Opening (0–3 sec)]
No greeting — state the point directly: "One thing that changed how I work:"
Or lead with the conclusion: "After using this method, I noticed…"

[Body (3 sec – 45 sec)]
One idea, explained clearly
Use a specific number or scenario, avoid vague generalizations

[Closing (last 5–8 sec)]
One-sentence takeaway
Optional CTA: "Worth sharing if this helped" / "Link in bio"
```

### The content pack five-piece set (assemble all five every time)

```
[Format recommendation]   Caption post or short video? Why?
[Content positioning]     Target audience / best distribution path
                          (Explore feed / subscriber feed / shares)
[Caption draft]           (produce this regardless of chosen format)
  - Opening line (most important observation / resonant thought)
  - Body
  - Closing + hashtags (Instagram) or description links (YouTube)
[Short video script draft] (if recommending video)
[Distribution advice]     Specific path into each platform's ecosystem
                          (Explore / Shorts shelf / Stories share / DM)
[Self-check verdict]      Does this make you want to share it with someone specific?
                          If not — what's missing?
```

### Distribution logic

The most effective paths on each platform:

**Instagram Reels:**
```
Post Reel → Share to Stories → Encourage DM shares → Explore discovery loop
Feed post (with Reel link) → Drive to Reel for deeper content
```

**YouTube Shorts:**
```
Upload Short → Appears in Shorts shelf and subscriber feed
Pin comment or description → Point to long-form companion video (Z04 content)
Long-form video → End-screen card to Short as a preview
```

If the owner already has a newsletter or blog (see Z01), suggest cross-promotion:
- Newsletter / blog post → link to accompanying Reel or Short in the email / post
- Short video → caption or description links back to full article

### Honest tooling note: the reality in 2026

Dedicated AI creation tools for Reels/Shorts are thin:
- No dedicated end-to-end workflow tool comparable to full-stack newsletter platforms
- No stable, developer-accessible publish API for either platform
- Most "automation" claims are workarounds that violate platform TOS

**What actually works:**
- Script writing: use content-matrix to adapt existing content into vertical video tone
- De-AI-ify: run through humanizer before finalizing
- Thumbnail / cover frame: use Canva (1:1 or 9:16 templates)
- Publishing: must be done manually — no stable AI automation available

This gap is a real opportunity — if the owner builds skills from this school's later modules, they may be able to contribute here.
