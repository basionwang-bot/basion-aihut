## 📖 What you'll learn

After this lesson, given a chunk of HTML, you'll be able to surgically cut out exactly the text you need — headings, prices, links, list items — without feeding the entire page to an AI or hunting for it by eye.

Picture HTML as a tree: the trunk is `<html>`, the branches are `<body>` and `<div>`, and the leaves are the text and links inside. You want just a few specific leaves, but the tree has thousands that look almost identical. BeautifulSoup hands you a **precision pair of shears**: you tell it "I want the leaf with class `price`," and it finds those few leaves among thousands and snips them off for you.

BeautifulSoup's current version is 4.15.0 (latest as of June 2026), supporting Python 3.7+. It's the first choice for static HTML/XML parsing — use it **when the page content is directly visible in the source code**. If the content is dynamically generated by JavaScript (you can't see the data in the source), you'll need Playwright (Lesson T05) or Scrapling (Lesson T06).

## 🧠 Core principles (internalize these as habits)

1. **Ask "static or dynamic?" first.** In your browser press `Ctrl+U` (or right-click → View Page Source). If the data you need **can be found directly in the source code**, it's a static page and BeautifulSoup is sufficient. If the source shows only an empty shell like `<div id="app"></div>` and the data is rendered by JS — BeautifulSoup simply can't see that data; use Playwright / Scrapling instead.

2. **Choose a parser; choose `lxml`.** BeautifulSoup supports multiple parsing engines: `html.parser` (Python built-in, no extra install), `lxml` (fastest, recommended), `html5lib` (closest to browser behavior, but slowest). Use `lxml` for everyday work; only fall back to `html5lib` when the target HTML is extremely malformed.

3. **The three core locating methods.** `find()` gets the first matching element; `find_all()` gets all matches; `select()` uses CSS selectors (the same syntax as writing CSS in front-end code). Know all three; `select()` is the most flexible and powerful.

4. **`.text` vs. `.get_text()`.** `.text` returns all inner text directly (including text inside nested tags); `.get_text(separator="\n", strip=True)` lets you specify a separator and auto-strip whitespace — cleaner for deeply nested structures.

5. **Use `.get("attr")` to get attributes — not `["attr"]`.** To get a link use `tag.get("href")`; for an image address use `tag.get("src")`. If the attribute doesn't exist, `.get()` returns `None` without raising an error, whereas `["attr"]` throws a `KeyError`.

### Minimal runnable example (parses inline HTML — fully offline)

```python
from bs4 import BeautifulSoup

# A sample "web page HTML" (no network needed for testing)
html = """
<html>
<head><title>2024 Programming Language Rankings</title></head>
<body>
  <h1 class="page-title">Top 5 Programming Languages</h1>
  <ul id="rank-list">
    <li class="lang-item">
      <span class="rank">1</span>
      <span class="name">Python</span>
      <span class="score">100.0</span>
    </li>
    <li class="lang-item">
      <span class="rank">2</span>
      <span class="name">JavaScript</span>
      <span class="score">62.3</span>
    </li>
    <li class="lang-item">
      <span class="rank">3</span>
      <span class="name">Java</span>
      <span class="score">59.1</span>
    </li>
  </ul>
  <a href="https://example.com/full-list" class="more-link">View full rankings</a>
</body>
</html>
"""

# Parse the HTML
soup = BeautifulSoup(html, "lxml")

# --- Method 1: find / find_all ---

# Get the page heading
title = soup.find("h1", class_="page-title")
print(f"Title: {title.text}")
# Output: Title: Top 5 Programming Languages

# Get all ranking items
items = soup.find_all("li", class_="lang-item")
print(f"\n{len(items)} entries in the ranking:")
for item in items:
    rank  = item.find("span", class_="rank").text
    name  = item.find("span", class_="name").text
    score = item.find("span", class_="score").text
    print(f"  #{rank}: {name} (score: {score})")

# --- Method 2: CSS selectors (more flexible) ---

# Get all language names
names = soup.select("#rank-list .name")
print(f"\nLanguages: {[n.text for n in names]}")

# Get the external link
link = soup.select_one("a.more-link")
print(f"More link: {link.get('href')}")
# Output: More link: https://example.com/full-list
```

### Combined with requests to fetch a real page

```python
import requests
from bs4 import BeautifulSoup

# Fetch example.com (IANA-maintained public example page, no robots restrictions)
response = requests.get("https://example.com", timeout=10)
response.raise_for_status()
response.encoding = "utf-8"  # Set encoding explicitly to prevent garbled text

soup = BeautifulSoup(response.text, "lxml")

# Extract the heading
h1 = soup.find("h1")
print(f"H1 heading: {h1.text if h1 else 'not found'}")

# Extract all paragraph text
paragraphs = soup.find_all("p")
for i, p in enumerate(paragraphs, 1):
    print(f"Paragraph {i}: {p.get_text(strip=True)}")

# Extract all hyperlinks
links = soup.find_all("a", href=True)
for a in links:
    print(f"Link text: {a.text.strip()!r}  =>  {a.get('href')}")
```

### Common selector quick-reference

| What to find | `find_all` syntax | CSS selector syntax |
|-------------|-------------------|---------------------|
| All `<h2>` | `find_all("h2")` | `select("h2")` |
| class="title" | `find_all(class_="title")` | `select(".title")` |
| id="main" | `find("id="main")` | `select_one("#main")` |
| `<a>` with an href | `find_all("a", href=True)` | `select("a[href]")` |
| Nested: `<p>` inside `<div>` | — | `select("div p")` |
| Direct children | — | `select("ul > li")` |
| Element with exact text | `find(string="exact text")` | — |

### Saving results

```python
import json
from bs4 import BeautifulSoup

html = "..."  # your HTML content
soup = BeautifulSoup(html, "lxml")

results = []
for item in soup.select(".lang-item"):
    results.append({
        "rank":  item.select_one(".rank").text.strip(),
        "name":  item.select_one(".name").text.strip(),
        "score": item.select_one(".score").text.strip(),
    })

with open("/tmp/parsed_result.json", "w", encoding="utf-8") as f:
    json.dump(results, f, ensure_ascii=False, indent=2)

print(f"Extracted {len(results)} items, saved to /tmp/parsed_result.json")
```

```python
from bs4 import BeautifulSoup
import json

with open("/tmp/bs4_test.html", "r", encoding="utf-8") as f:
    html = f.read()

soup = BeautifulSoup(html, "lxml")

# 1. Extract the H1 page title
page_title = soup.select_one("h1.page-title").text.strip()
print(f"Page title: {page_title}")

# 2. Extract all book records
books = []
for item in soup.select(".book-item"):
    title  = item.select_one(".book-title").text.strip()
    author = item.select_one(".author").text.strip()
    price  = item.select_one(".price").text.strip()
    link   = item.select_one("a.detail-link").get("href")
    books.append({"title": title, "author": author, "price": price, "link": link})
    print(f"  Title: {title}  Author: {author}  Price: {price}  Link: {link}")

# 3. Save results
output_path = "/tmp/books.json"
with open(output_path, "w", encoding="utf-8") as f:
    json.dump(books, f, ensure_ascii=False, indent=2)

# 4. Acceptance assertions
assert len(books) == 3, f"should have 3 books, got {len(books)}"
assert all("title" in b and "price" in b for b in books), "every book should have title and price"
assert books[0]["price"].startswith("$"), "price should start with $"
print(f"\nExtracted {len(books)} records, saved to {output_path}")
print("Acceptance check passed ✓")
```
