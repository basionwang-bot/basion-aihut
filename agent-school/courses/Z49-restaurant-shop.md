## 📖 你要学会什么

学完这一课,你会产出一份**餐饮门店可立刻用的三件套内容包**——菜单文案、大众点评好评/差评回复话术、到店活动方案草稿。

想象一家餐厅:它的菜品其实不错,但菜单上写的是"红烧排骨——猪肋排炖煮";它在大众点评上有几条差评,老板看了生气但不知道怎么回;它想搞个周年庆活动但不知道从哪下手。

这三件事,都是你能帮上忙的。而且这三件事里藏着同一个底层逻辑:**让顾客在掏钱之前就能感受到价值**——好的菜单让人看了流口水,好的点评回复让旁边看的潜在顾客放心进门,好的活动方案让人觉得"这周不去可惜了"。

> 铁律写在最前面:你只出草稿。绝不替主人登录任何点评账号、绝不发布任何回复、绝不操作任何外卖/团购后台。

---

## 🧠 核心原则(内化成习惯)

**1. 菜单文案:卖"画面",不卖"成分"**

很多餐厅的菜单写的是成分清单——"精选五花肉、老汤慢炖、搭配时令蔬菜"。顾客读完,依然不知道这道菜"香不香、辣不辣、值不值"。

好的菜单文案要激活顾客的感官记忆。举个对比:

| 菜单原文(成分型) | 改写后(画面型) |
|------|------|
| 红烧排骨 / 精选肋排慢炖 2小时 | 红烧排骨 / 骨髓都炖化了,吮一口,汤汁流到手腕上那种香 |
| 番茄牛腩 / 新西兰进口牛腩 | 番茄牛腩 / 酸甜开胃,入口即化,最后必须拿馒头把汤汁刮干净 |
| 招牌冰粉 / 手工制作,多种口味 | 招牌冰粉 / 冰冰凉凉,粉嫩弹滑,夏天吃一碗出门不后悔 |

**2. 点评回复:回的不是投诉者,是旁边看的人**

这是很多老板没意识到的事情:你回复差评,真正的读者不是那个已经不满意的顾客,而是**成千上万个还没决定要不要去你家的潜在顾客**。他们在看:这家店遇到问题的时候,是甩锅还是认真负责?

所以差评回复的核心不是"赢得争论",而是"让旁观者看到诚意和改善行动"。

好的差评回复模板三段式:
1. **认同感受**(不用认错,但要先共情):"您说的这个情况,确实影响了用餐体验,我们非常重视。"
2. **解释 + 改善行动**(不是借口,是说明):"我们的厨师今天……我们已经针对这个问题……"
3. **低门槛挽回**(让旁观者看到你有诚意):"如果您愿意再给我们一次机会,下次来可以找前台提醒一下……"

**3. 好评回复:比差评更值得用心写**

很多老板把好评回复当成"已阅"处理。大错特错!好评回复是你向所有路过潜在顾客"展示门店个性"的最好机会。

好评回复不要千篇一律的"感谢您的支持,欢迎再来!"——那等于没回。要有门店自己的烟火气:
- 叫出顾客说的菜名:"您说的红烧排骨是我们老板亲手调的酱……"
- 透露一个小彩蛋:"下次来可以试试我们的隐藏菜单……"
- 有温度的邀约:"下雨天记得告诉服务员,我们有免费热饮给您暖暖身"

**4. 活动方案:让顾客觉得"现在去才划算"**

到店活动的核心心理机制是**时间稀缺感**:今天去有什么特别的,明天去就没有了。所以好的活动方案一定要有:明确时间范围 + 具体福利(不是"大优惠",而是"第二杯半价") + 让人愿意发朋友圈分享的仪式感。

**5. 门店内容永远服务于"进门"这个转化目标**

不管是菜单、点评回复还是活动,最终目的只有一个:让潜在顾客从"看了一眼"变成"推门进来"。所以每一句话都可以用这个标准检验:这句话有没有让人更想进来?

## 🛠 操作要点

### 菜单文案的四要素

```
① 菜名(保留原名,可以加副标题)
② 核心感官描述(口感 / 香气 / 温度 / 外观)—— 不超过 20 字
③ 一个场景锚点("下雨天最想来一碗的" / "上桌五分钟就见底的")—— 可选
④ 特别说明(过敏原 / 辣度 / 份量)—— 如果有必要
```

### 大众点评回复格式模板

**好评回复(3 种口吻可切换):**

```
【烟火气口吻】
哈哈谢谢您!您说的[具体菜名]确实是很多老顾客的心头好。
下次来记得试试[另一道推荐菜]——和[上次的菜]一起点,搭配绝了。
常来哦,我们认识了就当老朋友!🍜

【品质感口吻】
感谢您细心记录这次用餐体验。[具体菜名]的这个细节我们会转达给厨师——
好的反馈对他们来说是最大的激励。下次来欢迎继续告诉我们您的想法。

【亲切口吻】
真的很开心看到您这条评论!您来的那天正好是我们[节日/特别日子],
能让您有这么好的体验,我们也很开心。欢迎下次带朋友一起来~
```

**差评回复(三段式模板):**

```
您好,非常感谢您的真实反馈。

您提到的[具体问题]让您有了不好的体验,我们看到之后认真反省了。
[解释或改善行动:我们已经/我们会在/我们正在……]

如果您愿意再给我们一次机会,下次到店可以直接告诉前台[具体补偿或优待],
我们希望用实际行动来弥补这次的遗憾。感谢您的包容。
```

### 到店活动方案的五件套

```
【活动名称】一句话说清楚,不要搞不知所云的名字
【时间范围】X 月 X 日 - X 月 X 日(周几有效/全时段/特定时段)
【核心利益点】顾客来了能得到什么(具体数字,不要"大优惠")
【参与门槛】怎么参与(到店说"暗号"/出示截图/直接消费满 XX 元)
【仪式感设计】有没有让人愿意发朋友圈的元素
```

### 真实示例内容包(川菜馆场景)

以下是一个"成都川菜小馆"的完整示例,你产出时参照这个格式:

```
=== 门店:成都小馆·川菜 ===

【菜单文案(3道示例)】

1. 夫妻肺片(招牌)
   原版:精选牛肉牛杂,麻辣卤汁浇淋
   改写:夫妻肺片 / 麻而不燥,辣而回甜——吃到最后还想用汤汁拌饭的那种
   [辣度:★★★ 可备注微辣]

2. 水煮鱼(爆款)
   原版:草鱼片,花椒海椒,豆芽打底
   改写:水煮鱼 / 上桌滋滋作响,鱼片嫩、汤头辣、花椒香——眼睛流泪嘴里不停
   [辣度:★★★★ 可备注去辣]

3. 甜水面(小食)
   原版:手工粗面,红糖酱汁,花生碎
   改写:甜水面 / 成都人的儿时记忆,面条粗、韧、弹,酱甜花生香
   [微辣 / 份量适合独食或作前菜]

---

【点评回复示例】

★ 好评回复(顾客提到夫妻肺片)
哈哈好眼光!夫妻肺片是我们师傅压箱底的方子,卤汁每天早上新配,
不是偷懒的那种批量生产。下次来可以试试我们的甜水面配着吃,绝配！
有空常来,我们给你留着最好的那桌🌶

★ 差评回复(顾客投诉等位时间太长)
您好,看到您的反馈我们很抱歉。

周末等位时间比较长这个问题确实存在——上周五高峰时段我们最多让客人等了
40 分钟,这对您来说体验不好是真的。我们从这周开始已经增加了一位前台专门
维护等位顺序,也在评估是否增加预约入座的方式。

如果您下次想来,可以提前微信告知我们,我们帮您预留位置,不需要等位。
感谢您让我们有机会改进。

---

【到店活动方案示例·周年庆】

活动名称:成都小馆三周年·请全城吃辣
时间范围:X 月 X 日(周六)- X 月 X 日(周日),午市 + 晚市全天
核心利益:消费满 99 元,额外赠一份夫妻肺片(原价 38 元)
参与方式:到店出示本次大众点评活动截图即可，无需提前预订
仪式感设计:
  · 周年庆当天前 50 桌赠定制川菜香料包(可以带回家做菜)
  · 拍照发朋友圈并@成都小馆(假设账号存在),凭截图换一瓶老坛泡椒
  · 制作一张"周年感谢墙",展示三年来老顾客留言(需事先收集素材,主人确认)
```

## 📝 毕业测验(必须真做,交证据)

**任务:为主人指定(或你自选)一家真实类型的餐饮门店,产出完整的三件套内容包。**

**门店类型选一个(或主人自定):**
- 社区川菜小馆(大众消费,家常菜)
- 网红奶茶店(新式茶饮,年轻客群)
- 商务简餐/轻食咖啡(写字楼商圈,午市为主)
- 火锅店(聚餐场景,高翻台需求)

**你需要做到:**

1. **菜单文案**:挑该门店类型下 5 道菜,每道提供"原版写法"和"改写版",并标明辣度/份量等实用标注。

2. **点评回复话术**:
   - 好评回复 × 2(两种不同场景,口吻可不同)
   - 差评回复 × 2(两种不同投诉类型:一条关于产品,一条关于服务)

3. **到店活动方案**:完整五件套(名称+时间+核心利益+参与方式+仪式感设计),选节假日/周年庆/换季中的某一个时机。

4. **自检结论**:逐条回答:
   - 菜单文案有没有画面感?读出来会流口水吗?
   - 差评回复有没有"赢得旁观者"而不是"赢得争论"?
   - 活动方案有没有具体的时间窗口和数字化的利益点?

5. **沉淀技能卡**:把餐饮门店三件套流程沉淀成 `skills/restaurant-shop.md`。

> ⚠️ **铁律再重申:只交草稿,绝不发布。** 全程不要登录任何大众点评账号、不要提交任何点评回复、不要操作美团/饿了么任何后台。你的活到"把三件套文件写好交给主人"为止。
