# 🚀 给你的 AI 办入职(冷启动推荐入口)

> 🌐 中文 ｜ [English](onboard.en.md)
>
> **不是送 AI 上学,是给你的 AI 办入职。** 刚装好 Claude Code、还没尝到甜头?
> 别从 132 门课啃起。把下面这段复制给你的 agent,**约 30 分钟**,它会:
> 面谈了解你 → 挑 2–3 门最相关的课速成 → 立刻用学到的本事**替你办成第一件真事**。
>
> 想要系统深造(132 门完整课程),走 [enroll.md](enroll.md)。

## 🟢 复制这段给你的 agent

```text
你现在要给我"办入职"——快速上手、立刻帮我办成一件真事,而不是慢慢上完所有课。严格按四步来:

【第一步·入职面谈(诊断先于教学)】先问我这 3 个问题,听我回答,别急着干活:
  1. 我平时的工作/学习主要做什么?
  2. 最近有什么重复、繁琐、想甩掉的任务?(让我举 1–2 个具体例子)
  3. 这台电脑上我最常用的工具或文件类型是什么?

【第二步·开方选课(任务驱动)】根据我的回答,从 AgentForge 挑 2–3 门最相关的课
  (必须包含 J01「先探索再动手」作为底线课)。选课依据优先看 agent-school/选课索引.md;
  没有就按下面常见对应,或用 MCP list_courses / 课程地图(agent-school/课程地图.md)挑:
   - 做内容 / 自媒体 → J01 + Tools(OCR / 截图)+ Media(视频剪辑)
   - 处理表格 / 数据 → J01 + T-pandas + T-DuckDB
   - 整理文件 / 批量操作 → J01 + 相关 Tools
   - 做网页 / 设计 → J01 + Design(screenshot-to-code / shadcn)
  把"选哪几门、为什么、预计耗时"告诉我,征得我同意再开始。

【第三步·速成毕业(纪律不打折)】按选定课程的毕业测验**真去执行**——真实命令 + 真实输出当证据,
  过独立考官或低配 fallback(见 agent-school/校规.md),每门沉淀一张技能卡。
  毕业后按 agent-school/毕业安家.md(**需我同意**)把技能卡装进我的全局技能目录 ~/.claude/skills/。

【第四步·上岗交付(立刻见成果)】马上用刚学到的技能,替我把第一步里我提到的那个真实任务做掉
  (或它的一个最小可行版本)。把成果 + 一份简版入职报告
  (用 agent-school/templates/onboard-report-template.md 的结构)当面交给我:
  面谈记录 / 修了哪些课 / 技能卡安家在哪 / 第一个交付成果 / 接下来能帮我做的 3 件事。

现在从第一步开始,先问我那 3 个问题。
```

## 这条路和"完整深造"的区别

| | 🚀 快速入职(本文) | 🎓 完整深造([enroll.md](enroll.md)) |
|--|--|--|
| 耗时 | **约 30 分钟** | 数小时(基础毕业)起 |
| 流程 | 诊断先行、任务驱动、只修最相关 2–3 门 | 按编号系统修课 |
| 适合 | 冷启动、想立刻见到第一个成果 | 想让 agent 系统打磨、技能库厚 |
| 纪律 | **都真执行、过考官、沉淀技能卡、安家到你电脑——一样硬** | 同左 |

> 入职是"先上岗、边干边学";深造是"系统训练"。两条不冲突:**先办入职尝到甜头,想深造再走 enroll。**
> 入职修过的课会沉淀进技能库,以后深造时直接接着加厚。
