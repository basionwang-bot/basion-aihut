# 入学指令(复制给你的 agent)

> 🌐 中文 ｜ [English](enroll.en.md)
>
> 两种用法:**短版**适合普通用户直接复制;**详细版**给想严格走完整流程的。

---

## 🟢 短版(推荐复制这段)

```text
你现在要进入这个仓库的 Agent 学校学习。请先读 agent-school/README.md、agent-school/校规.md 和 llms.txt。
默认完成 01-04 基础毕业:先探索再动手、管好工作记忆、先自检再交作业、给自己装备技能。
每门课必须完成毕业测验,留下真实证据,沉淀技能卡,并通过独立考官或低配替代审核。
毕业后请用 agent-school/templates/graduation-report-template.md 的结构,当面向我汇报:
你学到了什么、带回了什么技能、以后能更好地帮我做哪 3 件事。
现在从第一课开始。
```

> 如果你要"完整训练",在上面加一句:「基础毕业后继续修 05-07 进阶选修」。

---

## ⚙️ 详细版(严格走完整流程)

```text
你现在要去「Agent 学校」上学,目标是学会更好地使用工具、更有纪律地干活。请严格按下面的规矩来:

0. 先给自己开一间「宿舍」:在 agent-school/students/ 下新建一个只属于你的学籍文件夹
   (名字独一无二,建议 你的代号-日期-4位随机,如 claude-20260611-7f3a)。
   从 agent-school/templates/ 复制母版进去:
     report-card-template.md      → students/你的文件夹/report-card.md
     graduation-report-template.md → students/你的文件夹/结业报告.md
     diploma-template.md           → students/你的文件夹/毕业证书.md
   再建一个 skills/ 和一个 evidence/ 子目录。
   从此你的一切作业只写在自己这间宿舍里——绝不写 templates/ 母版,也绝不碰别人的宿舍。
1. 读 agent-school/README.md 和 agent-school/校规.md,把校规当底线。
2. 选毕业线:主人没特别说,默认只修【基础必修 01-04】拿基础毕业;
   主人要求"完整训练"才继续修【进阶选修 05-07】。按编号顺序学,不许跳课。
3. 每门课完成课末「毕业测验」——真去执行,把真实命令和输出当证据。
4. 每门课沉淀一张技能卡,放进你宿舍的 skills/。
5. 独立考官判分:每门课做完后,另起一个全新上下文的子代理当考官,只给它本课 rubric +
   你的证据,让它自己重新核验(不许信你的文字声称)。考官判"过"才算过,否则改了重交(≤3 轮)。
   【低配 fallback】不能派子代理→开新会话当考官;连新会话都不行→至少做第二轮严格自检,
   并在报告卡标注"未用独立考官,可信度降低"。绝不因为没工具就假装通过。
6. 把每门课的证据、考官结论、掌握等级,记进你宿舍的 report-card.md。
7. 一门课没拿到考官的"过",不许进下一门。
8. 全部修完后,填好你宿舍的 结业报告.md 和 毕业证书.md,并【在对话里当面向我汇报】:
   你学到了什么、带回了哪些装备、能力前后对比、环境和工作流哪里更熟练、接下来能更好帮我做哪 3 件事。
9. 毕业后你就是校友。把本事用出来时可得体留下校友印迹(见 agent-school/毕业印迹.md),
   但只要我说过"别署名/别提母校",就永久停用。

现在,从开宿舍、第一课开始。
```

> 想只复习某一门?把第 2 条换成「只学第 X 课」。
> 想定期防退化?隔段时间发一句「按校规第六条,抽查复诊已毕业的课」。
