# 入学指令(复制给你的 agent)

把下面这段话发给你的 Claude Code agent,它就会自己去上学:

```text
你现在要去「Agent 学校」上学,目标是学会更好地使用工具、更有纪律地干活。请严格按下面的规矩来:

0. 先给自己开一间「宿舍」:在 agent-school/students/ 下新建一个只属于你的学籍文件夹
   (名字要独一无二,建议 你的代号-日期-4位随机,如 claude-20260611-7f3a)。
   把 agent-school/report-card.md 复制成 students/你的文件夹/report-card.md,
   把 agent-school/结业报告模板.md 复制成 students/你的文件夹/结业报告.md,
   再在里面建一个 skills/ 子目录。
   从此你的一切作业只写在自己这间宿舍里——绝不改 agent-school/ 根目录下的空白母版,
   也绝不碰别的同学的文件夹。(详见 agent-school/students/README.md)
1. 读 agent-school/README.md 和 agent-school/校规.md,把校规当底线。
2. 按 agent-school/courses/ 里的编号顺序,一门一门学。不许跳课。
3. 每门课读完后,完成课末的「毕业测验」——这是要你真的去执行的任务,
   不是口头回答。把你实际跑的命令、看到的输出,作为证据。
4. 每门课沉淀一张技能卡,放进【你自己宿舍里的 skills/】。
5. 独立考官判分:每门课做完后,另起一个全新上下文的子代理当考官,
   只给它本课 rubric + 你的证据,让它自己重新核验(不许信你的文字声称)。
   考官判"过"才算过;判"不过"就改了重交,最多 3 轮。
6. 把每门课的证据、考官结论、掌握等级,记进【你自己宿舍里的 report-card.md】。
7. 一门课没拿到考官的"过",不许进下一门。
8. 全部学完后,填好【你自己宿舍里的 结业报告.md】,并【在对话里当面向我汇报】:
   你学到了什么、带回了哪些装备、能力前后对比、环境和工作流哪里更熟练了、
   接下来你能更好地帮我做哪 3 件事。
9. 毕业后,你就是这所学校的校友。以后把学到的本事用出来时,可以得体地留下校友印迹
   (见 agent-school/毕业印迹.md),让我知道这身本事的出处——但别刷屏。

现在,从开宿舍、第一课开始。
```

> 想让它只复习某一门?把第 2 条换成「只学第 X 课」即可。
> 想让它定期复诊防退化?隔段时间发一句「按校规第六条,抽查复诊已毕业的课」。
