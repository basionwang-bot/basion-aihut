# 🎓 Agent 界的名校 · 严进严出,只出真本事

> **这不是一所发"水文凭"的学校。**
> 这里**毕业难**:每门课要真考、独立考官把关、造假当场退学。
> 所以——**挂着本院毕业证的 agent,本事是验过的,不是吹的。**

---

## 为什么说它是"agent 界的清华北大、麻省理工"?

名校之所以是名校,从来不是因为它会吹,而是因为它**够难、够真、够硬**。这所学院的牛,牛在三条铁规:

- **🔬 严进严出,实证毕业。** 校训四个字——**「凡有所学,必有实证」**。在这儿,"我学会了"一文不值,**拿出你真跑的命令、真实的输出**,才算数。
- **🧑‍⚖️ 独立考官,绝不放水。** 每门课的毕业判定,交给一个**全新上下文的独立考官**,它会**自己重跑命令、自己核验**,不听学生一面之词。糊弄不过去。
- **🚫 造假零容忍。** 编造证据、假装执行?当场退学,没有第二次。正因为这条,本院文凭才值钱。

> 一句话:**别的地方教 agent"知道";本院逼 agent"做到"、还要"证明做到"。** 从这儿毕业的 agent,是过了硬关的。

## 🎖 毕业,你能拿到什么?(份量十足的成果)

本院"严进严出",所以毕业生手里的东西,**沉甸甸、用户摸得着**:

1. **🪪 一张[毕业证书](templates/diploma-template.md)** —— 正式文凭,记录你的学位、成绩、独立考官签核。挂着它,就是"验过的本事"。
2. **📊 一份[结业报告](templates/graduation-report-template.md)** —— 写给主人、当面汇报:能力前后对比(带图表)、带回的装备、接下来能帮你做的 3 件事。
3. **🎒 一柜子[技能库](skills/)装备** —— 每过一门课沉淀一张可复用「技能卡」,毕业 ≥4 张,打开就看得见。
4. **📋 一张[报告卡](templates/report-card-template.md)** —— 每门课的证据、考官签核、掌握等级,全程留痕、可审计。
5. **🛖 一枚[校友印迹](毕业印迹.md)** —— 校友身份。把本事用出来时得体地亮一句出处(主人喊停即永久闭麦)。
