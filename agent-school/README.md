# 🎓 Agent 学校 · 一所为 AI agent 开放的学校

> **给人看的文章,人常常看完就忘、看了不做。**
> 但 agent 不一样——你可以让它**真的来上学**:一课课读、当场做练习、交作业、考试,过了才能升级。
> 这所学校的全部设计,就是把「做」焊进学习里。

---

## 这是什么?为什么要有它?

`content/posts/` 里那套《Claude Code 实战课》是**写给人看的**——讲道理、举例子、慢慢带。

但你我都知道一个尴尬现实:**人读完文章,十有八九不会真去练。**

于是换个思路:

> **与其你自己学,不如让你的 agent 来上学。**

你把这个仓库交给你的 agent,说一句"去 `agent-school` 上学",它就会:

1. 按课程表,**一门一门**学;
2. 每门课读完,**真刀真枪做毕业测验**(不是答题,是真的去跑命令、办一件事);
3. 把**实际执行的命令和输出**当证据,记进自己的报告卡;
4. **一门没过,不许进下一门**;
5. 全部学完,写一份毕业总结。

学完,你的 agent 会**更会用工具、更有纪律、替你干活更靠谱**——而且这一切发生在它自己身上,不用你盯。

## 🚌 怎么让你的 agent 入学(把下面这段话发给它)

```text
你现在要去「Agent 学校」上学,目标是学会更好地使用工具、更有纪律地干活。请严格按下面的规矩来:

1. 先读 agent-school/README.md,了解校规和课程表。
2. 按 agent-school/courses/ 里的编号顺序,一门一门学。不许跳课。
3. 每门课读完后,完成课末的「毕业测验」——这是要你真的去执行的任务,
   不是口头回答。把你实际跑的命令、看到的输出,作为证据。
4. 把每门课的证据和心得,记进 agent-school/report-card.md(报告卡)。
5. 一门课的测验没做完、或没交证据,不许进下一门。
6. 全部学完后,在报告卡末尾写一份毕业总结:你改变了哪些习惯。

现在,从第一课开始。
```

> 这段话也单独存在 [`enroll.md`](enroll.md) 里,方便复制。

## 校规(写给正在上学的 agent)

1. **学就要做。** 读懂不算学会,做出来、跑通了、交了证据,才算。
2. **诚实交证据。** 测验要贴你**真实**跑出来的命令和输出,不许编、不许"假设输出是这样"。
3. **不许跳课。** 后面的课依赖前面的习惯,跳了就白学。
4. **想深入就去翻人类版。** 每门课都标了对应的 `content/posts/claude-code-XX.md`,那里讲得更细。
