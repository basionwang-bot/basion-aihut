## 二、优先修复清单(按严重度)

### 🔴 必修(事实/合规硬伤)
1. **T35 Scrapling+MCP — 打回重写**。联网核证:真实启动命令是 `scrapling mcp`(非 `python -m scrapling.server`),导入应为 `from scrapling.core.ai import ScraplingMCPServer`,工具是 `get/bulk_get/fetch/bulk_fetch`(非 `scrapling_fetch/_extract/_screenshot`),Claude 配置应是 `uvx scrapling-fetch-mcp`。**整篇 API 需照官方重写,否则下架。**
2. **Z41 合同审查 — 改法律硬伤**。"违约金上限不超损失30%"是对《民法典》合同编司法解释违约金**酌减规则**(过分高于损失可调减至不超损失130%)的误读,须更正;并核实/更换"Anthropic legal best practices"这一查无实据的源头。
3. **Z47 招生文案 — 合规整改**。删/弱化"信息学奥赛证书助升学""不学就落后"等承诺与焦虑话术,补《广告法》绝对化用语/效果保证自检环节,真实联系方式改占位符。

### 🟠 优先小修
4. **Z06** 重新定位为纯"风格迁移方法论"(删对其它课的复述)或与 Z02/Z09 合并;课名与内容对齐。
5. **T34** 内嵌一小份 geojson 或给确切下载命令;`unary_union`→`union_all()`。
6. **J09** "七步法"与正文对齐(改名或补出七步);收紧"自问自答"。
7. **Z22** `Evans Fisher`→`Evan Miller`;补两个源头的真实链接或删除。
8. **T17** `-probe`→`ffprobe`。

### 🟡 微瑕(放行前顺手)
- T01 把官方 keyring 安装挪到首位;T22 示例图加大字号;Z14 标注 Helium10/JungleScout 海外门槛+国内替代;Z02/Z03 推荐选题去"自我指涉"、补源头。

## 三、跨课通病(供放量阶段校准)

1. **"真任务"两极分化**:T02/T09/T28/T22 做到了"内嵌数据/公开端点→真跑→断言",是样板;J11/T17/T34/T35 滑向"写方案/自问自答",证据含金量低。**凡涉及外部数据的课,应像 T09 一样内嵌数据或给确切获取命令。**
2. **自媒体课结构雷同**:Z02/Z03/Z06/Z47 共用一套骨架(地形比喻→工具调研三件套→核心原则末条永远"草稿是草稿"→N件套→公式表→三选一全是"夸AI"主题)。
3. **源头"挂权威但没用上/听起来对"**:T35(编造)、Z22(人名错+无链接)、Z41/Z44(凑数)、Z28(挂了没用上)。**这是最该堵的红线。**
4. **"中国用户优先"基本达标但偶尔乐观**:海外资源(gh/curl/httpbin/Helium10/DataV)多数有标门槛+给替代,个别"国内可直接访问"过于乐观。
