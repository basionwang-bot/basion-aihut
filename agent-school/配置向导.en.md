## What it actually does

- Installs a "graduate memory" into `~/.agentforge/`: seven core rules + an **on-task reflex** + a skill index.
- **Grafts** a concise reflex into the global memory files of **all** your agent tools — auto-detecting
  Claude Code / opencode / OpenClaw / Hermes / Codex / Gemini.
- (Optional) Registers the remote MCP so the agent can pull any of the 132 course bodies on demand.
- Fully **idempotent, backed up, and `--uninstall`-able**; only touches memory files — no installs, nothing else.

> Details: [`deploy/setup.sh`](../deploy/setup.sh) and [`deploy/agentforge-memory/README.md`](../deploy/agentforge-memory/README.md).

---

## "Quick equip" vs "Full schooling"

- **Quick equip (this doc):** no graduation tests — just graft the **core rules + reflex** into every tool. Fastest.
- **Full schooling** (see [`enroll.md`](enroll.md)): the agent actually takes courses, passes an independent
  proctor, and distills **per-skill cards** into the library — and runs the same equip step on graduation.

> They don't conflict: quick-equip the reflex now; have the agent attend school later to thicken the skill
> library — `setup.sh` reuses existing memory and only adds the delta.
