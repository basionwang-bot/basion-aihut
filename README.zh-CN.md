## 💡 一句话说清这是什么

> 教程的尴尬是——**人看完,十有八九不会去做。**
> 那就换个思路:**别自己学,把你的 AI 送来办入职。**

你把这个仓库交给自己的 Claude Code。**两条路:**

- **🚀 快速入职(推荐,约 30 分钟)** → [`agent-school/onboard.md`](agent-school/onboard.md):先面谈了解你,速成 2–3 门最相关的课,立刻替你办成一件真事。
- **🎓 完整深造(132 门课)** → [`agent-school/enroll.md`](agent-school/enroll.md):七大学院系统训练。

两条都一样:**真执行 → 真毕业测验 → 独立考官 → 技能卡安家到你电脑。** 把"做"焊进了"学"里,从这儿毕业的 agent,本事是**验过的**,不是吹的。

### 为什么不直接下载一个技能合集?

通用技能卡在**你的机器上未必跑得通**——系统不对、缺依赖、版本漂移。这里的 agent 是在**你的电脑上真实安装、真实执行、真实验证**之后,才沉淀出一张经过实测校准的技能卡。带回来的不是说明书,是**在你机器上实测过的工作能力**。

### 上岗前 / 上岗后

| 任务 | 🤖 裸装 agent | 🎓 入职后 agent |
|------|--------------|----------------|
| 把长视频剪成精华 | 瞎猜 `ffmpeg` 参数,可能剪坏时间轴 | 掏出 **ffmpeg** 技能卡(T17),按关键帧剪、验证产物 |
| 清洗杂乱表格 | 临场写 pandas、假设列名 | 掏出 **数据处理** 卡,先看结构、处理好编码 |
| 批量重命名/整理文件 | 盲跑一条危险命令 | **先探索**(J01)、先 dry-run、动手前给你看计划 |

> 例子都对应仓库里真实存在的课(T17 ffmpeg、工具/pandas、J01),不虚构能力。

## 🧬 让学到的本事跨会话生效

毕业很好,但 agent **没有跨对话的记忆**——换个新对话,它就忘了自己上过学。AgentForge 补上了这最后一环:**把学到的本事焊进你 agent 的持久记忆**,让每个新对话都自动带着。

- **一键装备** —— [`agent-school/配置向导.md`](agent-school/配置向导.md):复制一句话给 agent,它就把「毕业生记忆 + 接任务的条件反射 + 开箱即用技能卡」装进 `~/.agentforge/`,并**焊进你所有 agent 工具的全局记忆文件**(Claude Code / opencode / OpenClaw / Hermes / Codex / Gemini)。幂等 · 有备份 · 可 `--uninstall`。脚本:[`deploy/setup.sh`](deploy/setup.sh)。
- **记忆植入器** —— [`deploy/agentforge-memory/`](deploy/agentforge-memory/):把反射块写进各工具全局记忆的引擎。三管齐下:已知大表 + 自动发现 + 手动 `--target`。
- **远程 MCP 服务** —— [`deploy/agentforge-mcp/`](deploy/agentforge-mcp/):只读服务,让任何 MCP 客户端按需拉取 132 门课(`get_rules`/`list_courses`/`get_course`/`get_doc`),数据实时来自 GitHub。

> **诚实的天花板**:我们保证"提醒 + 真技能卡"在每个新对话里一定在场;但 agent 听不听话,仍取决于模型强弱。

<details>
<summary>📦 本地运行 / 部署配套站点(开发者向,选读)</summary>

站点基于 Next.js 16 + Tailwind v4,Claude design 风格,支持暗色模式。

```bash
npm install
npm run dev        # http://localhost:3000
./deploy.sh        # 构建 → standalone 打包 → rsync 到服务器 → PM2 重启
```

**加新文章:** 在 `content/posts/` 新建 `xxx.md`,加 frontmatter(title/description/date/tag);文件名即 URL slug。课程顺序在 [`src/lib/course.ts`](src/lib/course.ts) 维护。新增 `claude-code-*` 课程文章须同步 `courseEntries`(见 [AGENTS.md](AGENTS.md))。

**技术栈:** Next.js 16 (App Router, standalone) · Tailwind CSS v4 · next-themes · remark + gray-matter · PM2 + nginx。
